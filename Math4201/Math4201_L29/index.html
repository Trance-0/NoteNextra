Theorem of uncountable compact Hausdorff spaces without isolated points

Any non-empty compact Hausdorff space XXX without any isolated points is uncountable.

Definition of limit point compact

A space XXX is limit point compact if any infinite subset of XXX has a limit point in XXX.

That is, ∀A⊆X\forall A\subseteq X∀A⊆X and AAA is infinite, there exists a point x∈Xx\in Xx∈X such that x∈Ux\in Ux∈U, ∀U∈T\forall U\in \mathcal{T}∀U∈T containing xxx, (U−{x})∩A≠∅(U-\{x\})\cap A\neq \emptyset(U−{x})∩A=∅.

This property also holds for finite sets, for example, any finite set with discrete topology is limit point compact. (since you cannot find a infinite subset of a finite set that has a limit point)

Definition of sequentially compact

A space XXX is sequentially compact if any sequence has a convergent subsequence. i.e. If {xn}n∈N\{x_n\}_{n\in\mathbb{N}}{xn​}n∈N​ is a sequence in XXX, then there are n1<n2<⋯<nk<…n_1<n_2<\dots<n_k<\dotsn1​<n2​<⋯<nk​<… such that {yi=xni}i∈N\{y_i=x_{n_i}\}_{i\in\mathbb{N}}{yi​=xni​​}i∈N​ is convergent.

Theorem of limit point compact spaces

If (X,d)(X,d)(X,d) is a metric space, then the following are equivalent:

  1. XXX is compact.
  2. XXX is limit point compact.
  3. XXX is sequentially compact.

Example of limit point compact spaces but not compact

Let X′={a,b}X'=\{a,b\}X′={a,b} with trivial topology, and X=N×X′X=\mathbb{N}\times X'X=N×X′ with the product topology where we use the discrete topology on N\mathbb{N}N.

Corollary of compact spaces

  1. If XXX is a compact topological space, then it is limit point compact.
  2. If XXX is a sequentially compact topological space, then it is limit point compact.

Proof

Proof of 1. follows from the theorem of limit point compact spaces.

That means, sequentially compact is a stronger property than limit point compact, and compact is the stronger property than limit point compact.
