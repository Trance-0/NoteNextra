Metric spaces

Basic properties and definitions

Definition of metric space

A metric space is a set XXX with a function d:X×X→Rd:X\times X\to \mathbb{R}d:X×X→R that satisfies the following properties:

  1. ∀x,y∈X,d(x,y)≥0\forall x,y\in X, d(x,y)\geq 0∀x,y∈X,d(x,y)≥0 and d(x,y)=0d(x,y)=0d(x,y)=0 if and only if x=yx=yx=y. (positivity)
  2. ∀x,y∈X,d(x,y)=d(y,x)\forall x,y\in X, d(x,y)=d(y,x)∀x,y∈X,d(x,y)=d(y,x). (symmetry)
  3. ∀x,y,z∈X,d(x,z)≤d(x,y)+d(y,z)\forall x,y,z\in X, d(x,z)\leq d(x,y)+d(y,z)∀x,y,z∈X,d(x,z)≤d(x,y)+d(y,z). (triangle inequality)

Definition of Metric topology

For any metric space (X,d)(X,d)(X,d), the topology generated by the balls of the metric space is called metric topology.

Definition of metrizable

A topological space (X,T)(X,\mathcal{T})(X,T) is metrizable if it is the metric topology for some metric ddd on XXX.

Q: When is a topological space metrizable?

Lemma: Every metric topology is Hausdorff

If a topology isn’t Hausdorff, then it isn’t metrizable.

Example of non-metrizable space

Trivial topology with at least two points is not Hausdorff, so it isn’t metrizable.

Finite complement topology on infinite set is not Hausdorff.

Suppose there exists x,y∈Xx,y\in Xx,y∈X such that x≠yx\neq yx=y and x∈U⊆Xx\in U\subseteq Xx∈U⊆X and y∈V⊆Xy\in V\subseteq Xy∈V⊆X such that X−UX-UX−U and X−VX-VX−V are finite.

Since U∩V=∅U\cap V=\emptysetU∩V=∅, we have V⊆X−UV\subseteq X-UV⊆X−U, which is finite. So X−VX-VX−V is infinite. (contradiction that X−VX-VX−V is finite)

So XXX with finite complement topology is not Hausdorff, so it isn’t metrizable.

Proof
