Definition of connected space

Let XXX be a topological space. XXX is separated if there exist two disjoint nonempty open subsets U,V⊂XU,V\subset XU,V⊂X such that X=U∪VX=U\cup VX=U∪V.

If XXX is not separated, then XXX is connected.

Definition of path-connected space

A topological space XXX is path-connected if for any two points x,x′∈Xx,x'\in Xx,x′∈X, there is a continuous map γ:[0,1]→X\gamma:[0,1]\to Xγ:[0,1]→X such that γ(0)=x\gamma(0)=xγ(0)=x and γ(1)=x′\gamma(1)=x'γ(1)=x′. Any such continuous map is called a path from xxx to x′x'x′.

Note

Path-connectedness is a stronger condition than connectedness.
