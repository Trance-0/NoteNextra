Proposition of second countable spaces

Let XXX be a second countable topological space. Then the following holds:

  1. Any discrete subspace YYY of XXX is countable
  2. There exists a countable subset of XXX that is dense in XXX (also called separable spaces)
  3. Every open covering of XXX has countable subcover (That is if X=⋃α∈IUαX=\bigcup_{\alpha\in I} U_\alphaX=⋃α∈I​Uα​, then there exists a countable subcover {Uα1,...,Uα∞}\{U_{\alpha_1}, ..., U_{\alpha_\infty}\}{Uα1​​,...,Uα∞​​} of XXX) (also called Lindelof spaces)

Proof

First we prove that any discrete subspace YYY of XXX is countable.

Let YYY be a discrete subspace of XXX. In particular, for any y∈Yy\in Yy∈Y we can find an element ByB_yBy​ of the countable basis B\mathcal{B}B for YYY such that By∩Y={y}B_y\cap Y=\{y\}By​∩Y={y}.

In particular, if y≠y′y\neq y'y=y′, then By≠By′B_y\neq B_{y'}By​=By′​. Because {y}=By∩Y≠By′∩Y={y′}\{y\}=B_y\cap Y\neq B_{y'}\cap Y=\{y'\}{y}=By​∩Y=By′​∩Y={y′}.

This shows that {By}y∈Y⊆B\{B_y\}_{y\in Y}\subseteq B{By​}y∈Y​⊆B has the same number of elements as YYY.

So YYY has to be countable.

Next we prove that there exists a countable subset of XXX that is dense in XXX.

For each basis element B∈BB\in \mathcal{B}B∈B, we can pick an element x∈Bx\in Bx∈B and let AAA be the union of all such xxx.

We claim that AAA is dense.

To show that AAA is dense, let UUU be a non-empty open subset of XXX.

Take an element x∈Ux\in Ux∈U. Note that by definition of basis, there is some element B∈BB\in \mathcal{B}B∈B such that x∈Bx\in Bx∈B. So x∈B∩Ux\in B\cap Ux∈B∩U. U∩B≠∅U\cap B\neq \emptysetU∩B=∅, so A∩U≠∅A\cap U\neq \emptysetA∩U=∅.

Since A∩U≠∅A\cap U\neq \emptysetA∩U=∅ this shows that AAA is dense.
