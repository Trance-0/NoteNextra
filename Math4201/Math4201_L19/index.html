Proposition for continuous and quotient maps

Let X,Y,ZX,Y,ZX,Y,Z be topological spaces. ppp is a quotient map from XXX to YYY and ggg is a continuous map from XXX to ZZZ.

Moreover, if for any y∈Yy\in Yy∈Y, the map ggg is constant on p−1(y)p^{-1}(y)p−1(y), then there is a continuous map f:Y→Zf: Y\to Zf:Y→Z satisfying f∘p=gf\circ p=gf∘p=g.

Proof

For any y∈Yy\in Yy∈Y, take x∈Xx\in Xx∈X such that p(x)=yp(x)=yp(x)=y (since ppp is surjective).

Define f(y)≔g(x)f(y)\coloneqq g(x)f(y):=g(x).

Note that this is well-defined and it doesn’t depend on the specific choice of xxx that p(x)=yp(x)=yp(x)=y because ggg is constant on p−1(y)p^{-1}(y)p−1(y).

Then we check that fff is continuous.

Let U⊆ZU\subseteq ZU⊆Z be open. Then we want to show that f−1(U)⊆Yf^{-1}(U)\subseteq Yf−1(U)⊆Y is open.

Since ppp is a quotient map, this is equivalent to showing that p−1(f−1(U))⊆Xp^{-1}(f^{-1}(U))\subseteq Xp−1(f−1(U))⊆X is open. Note that p−1(f−1(U))=g−1(U)p^{-1}(f^{-1}(U))=g^{-1}(U)p−1(f−1(U))=g−1(U).

Since ggg is continuous, g−1(U)g^{-1}(U)g−1(U) is open in XXX.

Since g−1(U)g^{-1}(U)g−1(U) is open in XXX, p−1(g−1(U))p^{-1}(g^{-1}(U))p−1(g−1(U)) is open in YYY.

In general, p−1(y)p^{-1}(y)p−1(y) is called the fiber of ppp over yyy. The ggg must be constant on the fiber.
