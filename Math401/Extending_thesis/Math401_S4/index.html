Complex Manifolds

For this week [10/21/2025], our goal is to understand the Riemann-Roch theorem and its applications.

Holomorphic vector bundles

Definition of real vector bundle

Let MMM be a topological space, A real vector bundle over MMM is a topological space EEE together with a surjective continuous map π:E→M\pi:E\to Mπ:E→M such that:

  1. For each p∈Mp\in Mp∈M, the fiber Ep=π−1(p)E_p=\pi^{-1}(p)Ep​=π−1(p) over ppp is endowed with the structure of a kkk-dimensional real vector space.
  2. For each p∈Mp\in Mp∈M, there exists an open neighborhood UUU of ppp and a homeomorphism Φ:π−1(U)→U×Rk\Phi: \pi^{-1}(U)\to U\times \mathbb{R}^kΦ:π−1(U)→U×Rk called a local trivialization such that:
    • π−1(U)=π\pi^{-1}(U)=\piπ−1(U)=π(where πU:U×Rk→π−1(U)\pi_U:U\times \mathbb{R}^k\to \pi^{-1}(U)πU​:U×Rk→π−1(U) is the projection map)
    • For each q∈Uq\in Uq∈U, the map Φq:Eq→Rk\Phi_q: E_q\to \mathbb{R}^kΦq​:Eq​→Rk is isomorphism from EqE_qEq​ to {q}×Rk≅Rk\{q\}\times \mathbb{R}^k\cong \mathbb{R}^k{q}×Rk≅Rk.

Definition of complex vector bundle

Let MMM be a topological space, A complex vector bundle over MMM is a real vector bundle EEE together with a complex structure on each fiber EpE_pEp​ that is compatible with the complex vector space structure.

  1. For each p∈Mp\in Mp∈M, the fiber Ep=π−1(p)E_p=\pi^{-1}(p)Ep​=π−1(p) over ppp is endowed with the structure of a kkk-dimensional complex vector space.
  2. For each p∈Mp\in Mp∈M, there exists an open neighborhood UUU of ppp and a homeomorphism Φ:π−1(U)→U×Ck\Phi: \pi^{-1}(U)\to U\times \mathbb{C}^kΦ:π−1(U)→U×Ck called a local trivialization such that:
    • π−1(U)=π\pi^{-1}(U)=\piπ−1(U)=π(where πU:U×Ck→π−1(U)\pi_U:U\times \mathbb{C}^k\to \pi^{-1}(U)πU​:U×Ck→π−1(U) is the projection map)
    • For each q∈Uq\in Uq∈U, the map Φq:Eq→Ck\Phi_q: E_q\to \mathbb{C}^kΦq​:Eq​→Ck is isomorphism from EqE_qEq​ to {q}×Ck≅Ck\{q\}\times \mathbb{C}^k\cong \mathbb{C}^k{q}×Ck≅Ck.

Definition of smooth complex vector bundle

If above MMM and EEE are smooth manifolds, π\piπ is a smooth map, and the local trivializations can be chosen to be diffeomorphisms (smooth bijections with smooth inverses), then the vector bundle is called a smooth complex vector bundle.

Definition of holomorphic vector bundle

If above MMM and EEE are complex manifolds, π\piπ is a holomorphic map, and the local trivializations can be chosen to be biholomorphic maps (holomorphic bijections with holomorphic inverses), then the vector bundle is called a holomorphic vector bundle.

Holomorphic line bundles

A holomorphic line bundle is a holomorphic vector bundle with rank 1.

Intuitively, a holomorphic line bundle is a complex vector bundle with a complex structure on each fiber.

Simplicial Cohomology

Riemann surfaces admit triangulations. The triangle are 2 simplices. The edges are 1 simplices. the vertices are 0 simplices.

Our goal is to build global description of Riemann surfaces using local description on each triangulation.
