Majorana stellar representation of quantum states

Tip

A helpful resource is Geometry of Quantum states  Section 4.4 and Chapter 7.

Vectors in Cn+1\mathbb{C}^{n+1}Cn+1 can be represented by a set of nnn degree polynomials.

Z⃗=(Z1,⋯ ,Zn)∼w(z)=Z0+Z1z+⋯+Znzn\vec{Z}=(Z_1,\cdots,Z_n)\sim w(z)=Z_0+Z_1z+\cdots+Z_nz^nZ=(Z1​,⋯,Zn​)∼w(z)=Z0​+Z1​z+⋯+Zn​zn

If Z0≠0Z_0\neq 0Z0​=0, then we can rescale the polynomial to make Z0=1Z_0=1Z0​=1.

Therefore, points in CPn\mathbb{C}P^{n}CPn will be one-to-one corresponding to the set of nnn degree polynomials with nnn complex roots.

Z0+Z1z+⋯+Znzn=0=Z0(z−z1)(z−z2)⋯(z−zn)Z_0+Z_1z+\cdots+Z_nz^n=0=Z_0(z-z_1)(z-z_2)\cdots(z-z_n)Z0​+Z1​z+⋯+Zn​zn=0=Z0​(z−z1​)(z−z2​)⋯(z−zn​)

If Z0=0Z_0=0Z0​=0, then count ∞\infty∞ as root.

Using stereographic projection of each root we can get a unordered collection of S2S^2S2. Example: CP=S2\mathbb{C}P=S^2CP=S2, Cp2=S2×S2∖S2\mathbb{C}p^2=S^2\times S^2\setminus S_2Cp2=S2×S2∖S2​ where S2S_2S2​ is symmetric group.
