Span and Linear Independence 2A

Recall

Proposition 2.22

In a vector space VVV, a spanning list {v1⃗,...,vn⃗}\{\vec{v_1},...,\vec{v_n}\}{v1​​,...,vn​​}, and an linearly independent list {w1⃗,...,wn⃗}\{\vec{w_1},...,\vec{w_n}\}{w1​​,...,wn​​}. Then m≤nm\leq nm≤n.

Definition 2.26

A list {v1⃗,...,vn⃗}\{\vec{v_1},...,\vec{v_n}\}{v1​​,...,vn​​} is called a basis if it is a linearly independent spanning list.

Proposition 2.ex.1

A subspace of a finite dimensional vector space is finite-dimensional.

Proof: Let VVV be a finite-dimensional vector space and let WWW be a subspace of VVV

  • Case 1: W={0⃗}W=\{\vec{0}\}W={0}

  • Case 2: Span{v1⃗,...,vk−1⃗}⊂WSpan\{\vec{v_1},...,\vec{v_{k-1}}\}\subset WSpan{v1​​,...,vk−1​​}⊂W where v1⃗,...,vk−1⃗\vec{v_1},...,\vec{v_{k-1}}v1​​,...,vk−1​​ is linearly independent

    If W=Span{v1⃗,...,vk−1⃗}W=Span\{\vec{v_1},...,\vec{v_{k-1}}\}W=Span{v1​​,...,vk−1​​}, done. If not, then there exists vk−1⃗∈W\vec{v_{k-1}}\in Wvk−1​​∈W and vk⃗∈Span{v1⃗,...,vk−1⃗}\vec{v_k}\cancel{\in} Span\{\vec{v_1},...,\vec{v_{k-1}}\}vk​​∈​Span{v1​​,...,vk−1​​}. This implies Span{v1⃗,...,vk⃗}⊂WSpan\{\vec{v_1},...,\vec{v_k}\}\subset WSpan{v1​​,...,vk​​}⊂W. and {v1⃗,...,vk⃗}\{\vec{v_1},...,\vec{v_k}\}{v1​​,...,vk​​} is linearly independent. Continue until Span{v1⃗,...,vn⃗}=W⊂VSpan\{\vec{v_1},...,\vec{v_n}\}=W\subset VSpan{v1​​,...,vn​​}=W⊂V, VVV has a finite spanning set,whose size ≥n\geq n≥n by Prop 2.22

Theorem 2.28

A list {v1⃗,...,vn⃗}\{\vec{v_1},...,\vec{v_n}\}{v1​​,...,vn​​} is a basis for VVV if and only if every vector v⃗∈V\vec{v}\in Vv∈V can be uniquely written as

v⃗=a1v1⃗+a2v2⃗+...+anvn⃗\vec{v}=a_1\vec{v_1}+a_2\vec{v_2}+...+a_n\vec{v_n}v=a1​v1​​+a2​v2​​+...+an​vn​​

Then 0⃗=(a1−b1)v1⃗+...+(an−bn)vn⃗\vec{0}=(a_1-b_1)\vec{v_1}+...+(a_n-b_n)\vec{v_n}0=(a1​−b1​)v1​​+...+(an​−bn​)vn​​

Lemma 2.30

Every Spanning set of a vector space can we be reduced into a basis.

ideas of Proof:

If the spanning list is not linearly independent, then use Lemma 2.19 to remove a vector.

Lemma 2.32

Every linearly independent list of vectors in a finite dimensional vector space can be extended with a basis.

ideas of Proof:

If {v1⃗,...,vk−1⃗}\{\vec{v_1},...,\vec{v_{k-1}}\}{v1​​,...,vk−1​​}, we can always add another vector vk⃗∈Span{v1⃗,...,vk−1⃗}\vec{v_k} \cancel{\in} Span\{\vec{v_1},...,\vec{v_{k-1}}\}vk​​∈​Span{v1​​,...,vk−1​​} to increase the span.

Theorem 2.31

Every finite dimensional vector space has a basis
