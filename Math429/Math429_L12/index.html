Definition 3.52

Let AAA be an m×nm \times nm×n matrix, then

  • The column rank of AAA is the dimension of the span of the columns in Fm,1\mathbb{F}^{m,1}Fm,1.
  • The row range of AAA is the dimension of the span of the row in F1,n\mathbb{F}^{1,n}F1,n.

Definition 3.58 Rank

The rank of a matrix AAA is the column rank of AAA denoted rank Arank\ Arank A.

Theorem 3.57

Given a matrix AAA the column rank equals the row rank.

Proof:

Note that by Theorem 3.56, if AAA is m×nm\times nm×n and has column rank ccc. A=CRA=CRA=CR for some CCC is a m×cm\times cm×c matrix, RRR is a c×nc\times nc×n matrices, ut the rows of CRCRCR are a linear combination of the rows of RRR, and row rank of R≤CR\leq CR≤C. So row rank A≤A\leqA≤ column rank of AAA.

Taking a transpose of matrix, then row rank of A⊤A^\topA⊤ (column rank of AAA) ≤\leq≤ column rank of A⊤A^\topA⊤ (row rank AAA).

So column rank is equal to row rank.
