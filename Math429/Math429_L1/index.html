Definition 1.20

  • Commutativity: ∀v,w∈V,v+w=w+v\forall v, w\in V,v+w=w+v∀v,w∈V,v+w=w+v
  • Associativity: ∀u,v,w∈V,(u+v)+w=u+(v+w)\forall u,v,w\in V,(u+v)+w=u+(v+w)∀u,v,w∈V,(u+v)+w=u+(v+w)

Theorem 1.26~1.30

Other properties of vector space

If VVV is a vector space on v∈V,a∈Fv\in V,a\in\mathbb{F}v∈V,a∈F

  • 0⋅v=00\cdot v=00⋅v=0
  • a⋅0=0a\cdot 0=0a⋅0=0
  • (−1)⋅v=−v(-1)\cdot v=-v(−1)⋅v=−v
  • uniqueness of additive identity
  • uniqueness of additive inverse

Example

Proof for 0⋅v=00\cdot v=00⋅v=0

Let v∈Vv\in Vv∈V be a vector, then (0+0)⋅v=0⋅v(0+0)\cdot v=0\cdot v(0+0)⋅v=0⋅v, using the distributive law we can have 0⋅v+0⋅v=0⋅v0\cdot v+0\cdot v=0\cdot v0⋅v+0⋅v=0⋅v, then 0⋅v=00\cdot v=00⋅v=0

Proof for unique additive identity

Suppose 000 and 0′0'0′ are both additive identities for some vector space VVV.

Then 0′=0′+0=0+0′=00' = 0' +0 = 0 +0' = 00′=0′+0=0+0′=0,

where the first equality holds because 000 is an additive identity, the second equality comes from commutativity, and the third equality holds because 0′0'0′ is an additive identity. Thus 0′=0' = 0′=0, proving that 𝑉 has only one additive identity.

Definition 1.22

Real vector space, complex vector space

  • A vector space over R\mathbb{R}R is called a real vector space.
  • A vector space over C\mathbb{C}C is called a complex vector space.

Example:

If F\mathbb{F}F is a vector space, prove that F2\mathbb{F}^2F2 is a vector space

We proceed by iterating the properties of the vector space.

For example, Existence of additive identity in F2\mathbb{F}^2F2 is (0,0)(0,0)(0,0), it is obvious that ∀(a,b)∈F2,(a,b)+(0,0)=(a,b)\forall (a,b)\in \mathbb{F}^2, (a,b)+(0,0)=(a,b)∀(a,b)∈F2,(a,b)+(0,0)=(a,b). Thus, (0,0)(0,0)(0,0) is the additive identity in F2\mathbb{F}^2F2.
