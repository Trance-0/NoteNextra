Lebesgue Measure

Caratheodory’s criterion:

SSS is Lebesgue measurable if for all A⊂SA\subset SA⊂S,

me(X)=me(X∩S)+me(X∩Sc)m_e(X) = m_e(X\cap S) + m_e(X\cap S^c)me​(X)=me​(X∩S)+me​(X∩Sc)

Let M\mathfrak{M}M be the collection of all Lebesgue measurable sets.

  1. ϕ∈M\phi\in\mathfrak{M}ϕ∈M
  2. M\mathfrak{M}M is closed under countable unions (proved last lecture)
  3. M\mathfrak{M}M is closed under complementation (M\mathfrak{M}M is a σ\sigmaσ-algebra) (goal today)

Desired properties of a measure:

  1. m(I)=ℓ(I)m(I)=\ell(I)m(I)=ℓ(I) for all intervals III
  2. If {Sn}n=1∞\{S_n\}_{n=1}^{\infty}{Sn​}n=1∞​ is a set of pairwise disjoint Lebesgue measurable sets, then

m(⋃n=1∞Sn)=∑n=1∞m(Sn) m\left(\bigcup_{n=1}^{\infty}S_n\right) = \sum_{n=1}^{\infty}m(S_n)m(⋃n=1∞​Sn​)=∑n=1∞​m(Sn​) 3. If R⊂SR\subset SR⊂S, then m(S∖R)=m(S)−m(R)m(S\setminus R) = m(S) - m(R)m(S∖R)=m(S)−m(R)

Recall the Borel σ\sigmaσ-algebra B\mathcal{B}B was the smallest σ\sigmaσ-algebra containing closed intervals. Therefore B⊂M\mathcal{B}\subset\mathfrak{M}B⊂M.

Theorem 5.9 (Finite union/intersection of Lebesgue measurable sets is Lebesgue measurable)

Any finite union/intersection of Lebesgue measurable sets is Lebesgue measurable.

Proof

Suppose S1,S2S_1, S_2S1​,S2​ is a measurable, and we need to show that S1∪S2S_1\cup S_2S1​∪S2​ is measurable. Given XXX, need to show that

me(X)=me(X1∪X2∪X3)+me(X4)m_e(X) = m_e(X_1\cup X_2\cup X_3)+ m_e(X_4)me​(X)=me​(X1​∪X2​∪X3​)+me​(X4​)

Since S1S_1S1​ measurable, me(X1∪X2∪X3)=me(X3)+me(X1∪X2)m_e(X_1\cup X_2\cup X_3)=m_e(X_3)+m_e(X_1\cup X_2)me​(X1​∪X2​∪X3​)=me​(X3​)+me​(X1​∪X2​).

Therefore,

me(X)=me(X1∪X2∪X3)+me(X4)=me(X1∪X2)+me(X3)+me(X4)=me(X1∪X2)+me(X3∪X4)=me(X)\begin{aligned} m_e(X) &= m_e(X_1\cup X_2\cup X_3) + m_e(X_4) \\ &= m_e(X_1\cup X_2) + m_e(X_3)+m_e(X_4) \\ &= m_e(X_1\cup X_2) + m_e(X_3\cup X_4) \\ &= m_e(X) \end{aligned}me​(X)​=me​(X1​∪X2​∪X3​)+me​(X4​)=me​(X1​∪X2​)+me​(X3​)+me​(X4​)=me​(X1​∪X2​)+me​(X3​∪X4​)=me​(X)​

Any countable union/intersection of Lebesgue measurable sets is Lebesgue measurable.

Corollary from the proof

Every open or closed set is Lebesgue measurable.

(Every open set is a countable union of disjoint open intervals)
