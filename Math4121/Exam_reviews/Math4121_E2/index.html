Extra terminology:

Dense:

A set SSS is dense in the interval III is every open subinterval of III contains a point of SSS.

This is equivalent to saying that SSS is dense in III if every point of III is a limit point of SSS or a point of SSS. (proved in homework)

Totally discontinuous:

A discontinuous function is totally discontinuous in an interval if the set of points of continuity is not dense in that interval.

In other words, there exists an open interval III such that the set of points of continuity of fff in III is empty.

Pointwise discontinuity:

A discontinuous function is pointwise discontinuous if the set of points of discontinuity is dense in the domain of fff.

Accumulation point (limit point):

A point ppp is an accumulation point of a set SSS if every neighborhood of ppp contains a point of SSS other than ppp itself. (That is, there exists a convergent sequence {pn}n=1∞\{p_n\}_{n=1}^\infty{pn​}n=1∞​ in SSS such that lim⁡n→∞pn=p\lim_{n\to\infty} p_n = plimn→∞​pn​=p and pn≠pp_n \neq ppn​=p for all n∈Nn \in \mathbb{N}n∈N. Proved in Rudin)

Derived set:

The derived set of a set SSS is the set of all accumulation points of SSS. S′={p∈R∣∀ϵ>0,∃x∈S s.t. 0<∣x−p∣<ϵ}S' = \{p \in \mathbb{R} \mid \forall \epsilon > 0, \exists x \in S \text{ s.t. } 0 < |x-p| < \epsilon\}S′={p∈R∣∀ϵ>0,∃x∈S s.t. 0<∣x−p∣<ϵ}.

A set SSS is of first species if it is type nnn for some n≥0n\geq 0n≥0, otherwise it is of second species.

Open set:

A set SSS is open if for every x∈Sx \in Sx∈S, there exists an ϵ>0\epsilon > 0ϵ>0 such that Bϵ(x)⊆SB_\epsilon(x) \subseteq SBϵ​(x)⊆S.

Closed set:

A set SSS is closed if its complement is open.

Equivalently, a set SSS is closed if it contains all of its limit points. That is S′⊆SS' \subseteq SS′⊆S.

Theorem 3.4

Bolzano-Weierstrass Theorem:

Every bounded infinite set has an accumulation point.

Proof:

Let SSS be a bounded infinite set. Cut the interval [a,b][a,b][a,b] into two halves, and let I1I_1I1​ be one with infinitely many points of SSS. (such set exists since SSS is infinite.)

Let I2I_2I2​ be the one half with infinitely many points of I1I_1I1​.

By induction, we can cut the interval into two halves, and let In+1I_{n+1}In+1​ be the one half with infinitely many points of InI_nIn​.

By the nested interval property, there exists a point ccc that is in all InI_nIn​.

ccc is an accumulation point of SSS.

QED

Theorem 3.6 (Heine-Borel Theorem)

For any open cover of a compact set, there exists a finite subcover.

Compact set:

A set SSS is compact if every open cover of SSS has a finite subcover. In R\mathbb{R}R, this is equivalent to being closed and bounded.

Cardinality:

The cardinality of R\mathbb{R}R is c\mathfrak{c}c.

The cardinality of N\mathbb{N}N, Z\mathbb{Z}Z, and Q\mathbb{Q}Q is ℵ0\aleph_0ℵ0​.

Chapter 4

Nowhere Dense set

A set SSS is nowhere dense if there are no open intervals in which SSS is dense.

That is equivalent to S′S'S′ contains no open intervals.

Note: If SSS is nowhere dense, then ScS^cSc is dense. But if SSS is dense, ScS^cSc is not necessarily nowhere dense. (Consider Q\mathbb{Q}Q)

Perfect Set

A set SSS is perfect if S′=SS'=SS′=S.

Example: open intervals, Cantor set.

Cantor set

The Cantor set (SVC(3)SVC(3)SVC(3)) is the set of all real numbers in [0,1][0,1][0,1] that can be represented in base 3 using only the digits 0 and 2.

The outer content of the Cantor set is 0.

The outer content of SVC(n)SVC(n)SVC(n) is n−3n−2\frac{n-3}{n-2}n−2n−3​.
