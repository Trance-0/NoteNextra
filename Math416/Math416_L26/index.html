Note: Contour can never go through a singularity.

Recall the semi annulus contour.

Know that ∫γf(z)dz=0\int_\gamma f(z)dz=0∫γ​f(z)dz=0.

So ∫A+∫B+∫C+∫D=0\int_A+\int_B+\int_C+\int_D=0∫A​+∫B​+∫C​+∫D​=0.

From last lecture, we know that ∫D=0\int_D=0∫D​=0 and ∫A+∫C=2i∫0∞sin⁡xxdx\int_A+\int_C=2i\int_0^\infty \frac{\sin x}{x}dx∫A​+∫C​=2i∫0∞​xsinx​dx.

∫B=−∫0πf(ϵeit)ϵieitdt\int_B=-\int_0^\pi f(\epsilon e^{it})\epsilon i e^{it}dt∫B​=−∫0π​f(ϵeit)ϵieitdt.

f(z)=eizz=1z(1+iz−z22!+⋯ )f(z)=\frac{e^{iz}}{z}=\frac{1}{z}(1+iz-\frac{z^2}{2!}+\cdots)f(z)=zeiz​=z1​(1+iz−2!z2​+⋯).

So zf(z)=1+O(ϵ)z f(z)=1+O(\epsilon)zf(z)=1+O(ϵ) and f(z)=1z+O(ϵz)f(z)=\frac{1}{z}+O(\frac{\epsilon}{z})f(z)=z1​+O(zϵ​).

∫B=−∫0π(1ϵeit+O(1))ϵieitdt=−i∫0π1dt+O(ϵ)=−iπ+O(ϵ)\begin{aligned} \int_B&=-\int_0^\pi (\frac{1}{\epsilon}e^{it}+O(1))\epsilon i e^{it}dt\\ &=-i\int_0^\pi 1dt+O(\epsilon)\\ &=-i\pi+O(\epsilon) \end{aligned}∫B​​=−∫0π​(ϵ1​eit+O(1))ϵieitdt=−i∫0π​1dt+O(ϵ)=−iπ+O(ϵ)​

Integrating over DDD

Method 1: Using estimate

z=Reitz=Re^{it}z=Reit for t∈[0,π]t\in[0,\pi]t∈[0,π].

f(z)=eizz=eiReitReitf(z)=\frac{e^{iz}}{z}=\frac{e^{iRe^{it}}}{Re^{it}}f(z)=zeiz​=ReiteiReit​.

Reit=R(cos⁡t+isin⁡t)Re^{it}=R(\cos t+i\sin t)Reit=R(cost+isint), iReit=−R(sin⁡t−icos⁡t)iRe^{it}=-R(\sin t-i\cos t)iReit=−R(sint−icost).

eiReit=e−Rsin⁡teiRcos⁡te^{iRe^{it}}=e^{-R\sin t}e^{iR\cos t}eiReit=e−RsinteiRcost.

max⁡∣f(z)∣=max⁡∣eiRcos⁡t∣∣Reit∣=1R\max|f(z)|=\max\frac{|e^{iR\cos t}|}{|R e^{it}|}=\frac{1}{R}max∣f(z)∣=max∣Reit∣∣eiRcost∣​=R1​.

This only bounds the function ∣∫D∣≤πR1R=π|\int_D|\leq \pi R\frac{1}{R}=\pi∣∫D​∣≤πRR1​=π.

This is not a good estimate.

Method 2: Hard core integration

γ(t)=Reit\gamma(t)=Re^{it}γ(t)=Reit for t∈[0,π]t\in[0,\pi]t∈[0,π].

∫D=∫0πeiReitReitiReitdt=i∫0πeiRcos⁡te−Rsin⁡tdt\begin{aligned} \int_D&=\int_0^\pi \frac{e^{iRe^{it}}}{R e^{it}}iR e^{it}dt\\ &=i\int_0^\pi e^{iR\cos t}e^{-R\sin t}dt\\ \end{aligned}∫D​​=∫0π​ReiteiReit​iReitdt=i∫0π​eiRcoste−Rsintdt​

Notice that we can use 2πt\frac{2}{\pi}tπ2​t to replace sin⁡t\sin tsint.

∣∫D∣≤∫0πe−Rsin⁡tdt=2∫0π/2e−Rsin⁡tdt≤2∫0π/2e−2Rt/πdt=−2πR(e−Rπ2t)∣0π/2≤πR\begin{aligned} \left|\int_D\right|&\leq\int_0^\pi e^{-R\sin t}dt\\ &=2\int_0^{\pi/2} e^{-R\sin t}dt\\ &\leq 2\int_0^{\pi/2} e^{-2Rt/\pi}dt\\ &=-\frac{2\pi}{R}(e^{-\frac{R\pi}{2}t})|_0^{\pi/2}\\ &\leq\frac{\pi}{R} \end{aligned}​∫D​​​≤∫0π​e−Rsintdt=2∫0π/2​e−Rsintdt≤2∫0π/2​e−2Rt/πdt=−R2π​(e−2Rπ​t)∣0π/2​≤Rπ​​

So ∫D=0\int_D=0∫D​=0.

So we have ∫A+∫C=2i∫0∞sin⁡xxdx=iπ\int_A+\int_C=2i\int_0^\infty \frac{\sin x}{x}dx=i\pi∫A​+∫C​=2i∫0∞​xsinx​dx=iπ.

So ∫0∞sin⁡xxdx=π2\int_0^\infty \frac{\sin x}{x}dx=\frac{\pi}{2}∫0∞​xsinx​dx=2π​.

Application to evaluate ∫−∞∞cos⁡x1+x4dx\int_{-\infty}^\infty \frac{\cos x}{1+x^4}dx∫−∞∞​1+x4cosx​dx

f(z)=eiz1+z4=cos⁡z+isin⁡z1+z4f(z)=\frac{e^{iz}}{1+z^4}=\frac{\cos z+i\sin z}{1+z^4}f(z)=1+z4eiz​=1+z4cosz+isinz​.

Our desired integral can be evaluated by ∫−RRf(z)dz\int_{-R}^R f(z)dz∫−RR​f(z)dz

To evaluate the singularity, z4=−1z^4=-1z4=−1 has four roots by the De Moivre’s theorem.

z4=−1=eiπ+2kπiz^4=-1=e^{i\pi+2k\pi i}z4=−1=eiπ+2kπi for k=0,1,2,3k=0,1,2,3k=0,1,2,3.

So z=eiθz=e^{i\theta}z=eiθ for θ=π4+kπ2\theta=\frac{\pi}{4}+\frac{k\pi}{2}θ=4π​+2kπ​ for k=0,1,2,3k=0,1,2,3k=0,1,2,3.

So the singularities are z=eiπ/4,ei3π/4,ei5π/4,ei7π/4z=e^{i\pi/4},e^{i3\pi/4},e^{i5\pi/4},e^{i7\pi/4}z=eiπ/4,ei3π/4,ei5π/4,ei7π/4.

Only z=eiπ/4,ei3π/4z=e^{i\pi/4},e^{i3\pi/4}z=eiπ/4,ei3π/4 are in the upper half plane.

So we can use the semi-circle contour to evaluate the integral. Name the path as γ\gammaγ.

∫γf(z)dz=2πi[Res⁡z=eiπ/4(f)+Res⁡z=ei3π/4(f)]\int_\gamma f(z)dz=2\pi i\left[\operatorname{Res}_{z=e^{i\pi/4}}(f)+\operatorname{Res}_{z=e^{i3\pi/4}}(f)\right]∫γ​f(z)dz=2πi[Resz=eiπ/4​(f)+Resz=ei3π/4​(f)].

The two poles are simple poles.

Res⁡z0(f)=lim⁡z→z0(z−z0)f(z)\operatorname{Res}_{z_0}(f)=\lim_{z\to z_0}(z-z_0)f(z)Resz0​​(f)=limz→z0​​(z−z0​)f(z).

So

Res⁡z=eiπ/4(f)=lim⁡z→eiπ/4(z−eiπ/4)eiz1+z4=(z−eiπ/4)eiz(z−eiπ/4)(z−ei3π/4)(z−ei5π/4)(z−ei7π/4)=eieiπ/4(eiπ/4−ei3π/4)(eiπ/4−ei5π/4)(eiπ/4−ei7π/4)\begin{aligned} \operatorname{Res}_{z=e^{i\pi/4}}(f)&=\lim_{z\to e^{i\pi/4}}(z-e^{i\pi/4})\frac{e^{iz}}{1+z^4}\\ &=\frac{(z-e^{i\pi/4})e^{iz}}{(z-e^{i\pi/4})(z-e^{i3\pi/4})(z-e^{i5\pi/4})(z-e^{i7\pi/4})}\\ &=\frac{e^{ie^{i\pi/4}}}{(e^{i\pi/4}-e^{i3\pi/4})(e^{i\pi/4}-e^{i5\pi/4})(e^{i\pi/4}-e^{i7\pi/4})} \end{aligned}Resz=eiπ/4​(f)​=z→eiπ/4lim​(z−eiπ/4)1+z4eiz​=(z−eiπ/4)(z−ei3π/4)(z−ei5π/4)(z−ei7π/4)(z−eiπ/4)eiz​=(eiπ/4−ei3π/4)(eiπ/4−ei5π/4)(eiπ/4−ei7π/4)eieiπ/4​​

A short cut goes as follows:

We know p(z)=1+z4p(z)=1+z^4p(z)=1+z4 has four roots z1,z2,z3,z4z_1,z_2,z_3,z_4z1​,z2​,z3​,z4​.

lim⁡z→z0(z−z0)p(z)=1p′(z0)\lim_{z\to z_0}\frac{(z-z_0)}{p(z)}=\frac{1}{p'(z_0)}z→z0​lim​p(z)(z−z0​)​=p′(z0​)1​

So

Res⁡z=eiπ/4(f)=eieiπ/44ei3π/4\operatorname{Res}_{z=e^{i\pi/4}}(f)=\frac{e^{ie^{i\pi/4}}}{4e^{i3\pi/4}}Resz=eiπ/4​(f)=4ei3π/4eieiπ/4​

Similarly,

Res⁡z=ei3π/4(f)=eiei3π/44eiπ/4\operatorname{Res}_{z=e^{i3\pi/4}}(f)=\frac{e^{ie^{i3\pi/4}}}{4e^{i\pi/4}}Resz=ei3π/4​(f)=4eiπ/4eiei3π/4​

So the sum of the residues is
