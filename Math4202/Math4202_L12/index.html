Fundamental group

Recall from last lecture, the (pi1(X,x0),∗)(pi_1(X,x_0),*)(pi1​(X,x0​),∗) is a group, and for any two points x0,x1∈Xx_0,x_1\in Xx0​,x1​∈X, the group (pi1(X,x0),∗)(pi_1(X,x_0),*)(pi1​(X,x0​),∗) is isomorphic to (pi1(X,x1),∗)(pi_1(X,x_1),*)(pi1​(X,x1​),∗) if x0,x1x_0,x_1x0​,x1​ is path connected.

Definition of simply connected

A space XXX is simply connected if

  • XXX is path-connected  (∀x0,x1∈X\forall x_0,x_1\in X∀x0​,x1​∈X, there exists a continuous function α:[0,1]→X\alpha:[0,1]\to Xα:[0,1]→X such that α(0)=x0\alpha(0)=x_0α(0)=x0​ and α(1)=x1\alpha(1)=x_1α(1)=x1​)
  • pi1(X,x0)pi_1(X,x_0)pi1​(X,x0​) is the trivial group for some x0∈Xx_0\in Xx0​∈X

Example of simply connected space

Intervals are simply connected.

Any star-shaped is simply connected.

S1S^1S1 is not simply connected, but n≥2n\geq 2n≥2, then SnS^nSn is simply connected.
