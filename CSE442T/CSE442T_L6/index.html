  2. Suppose it’s not hard to invert. Then ∃n.u.p.p.t. A\exists \text{n.u.p.p.t.}\ \mathcal{A}∃n.u.p.p.t. Asuch that P[w←{0,1}8n4;z=f(w):f(A(z))=0]=μ(n)>1p(n)P[w\gets \{0,1\}^{8n^4};z=f(w):f(\mathcal{A}(z))=0]=\mu (n)>\frac{1}{p(n)}P[w←{0,1}8n4;z=f(w):f(A(z))=0]=μ(n)>p(n)1​

function B:
    Receives N
    Sample (x,y) q times
    Compute z_i = f_mult(x_i,y_i) for each i
    From i=1 to q
        check if both x_i y_i are prime
        If yes,
            z_i = N

    Let z = (z_1,z_2,...,z_q) // z_k = N hopefully
    ((x_1,y_1),...,(x_k,y_k),...,(x_q,y_q)) <- a(z)
    if (x_k,y_k) was replaced
        return x_k,y_k
    else
        return null
