Euclidean algorithm

Recursively relying on fact that (a>b>0)(a>b>0)(a>b>0)

def euclidean_algorithm(a,b):
    if a<b: return euclidean_algorithm(b,a)
    if b==0: return a
    return euclidean_algorithm(b,a%b)

Extended Euclidean algorithm

Our goal is to find x,yx,yx,y such that ax+by=gcd(a,b)ax+by=gcd(a,b)ax+by=gcd(a,b)

Given a⋅x≡bmod  Na\cdot x\equiv b\mod Na⋅x≡bmodN, we do euclidean algorithm to find gcd(a,b)=dgcd(a,b)=dgcd(a,b)=d, then reverse the steps to find x,yx,yx,y such that ax+by=dax+by=dax+by=d

def extended_euclidean_algorithm(a,b):
    if a%b==0: return (0,1)
    x,y=extended_euclidean_algorithm(b,a%b)
    return (y,x-y*(a//b))
