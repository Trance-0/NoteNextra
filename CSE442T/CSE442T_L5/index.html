Chapter 2: Computational Hardness

Proving that there are one-way functions relies on assumptions.

Factoring Assumption: ∀A,∃ϵ(n)\forall \mathcal{A}, \exist \epsilon (n)∀A,∃ϵ(n), let p,q∈Πn,p,q<2np,q\in \Pi_n,p,q<2^np,q∈Πn​,p,q<2n

P[p←Πn;q←Πn;N=p⋅q:A(N)∈{p,q}]<ϵ(n)P[p\gets \Pi_n;q\gets \Pi_n;N=p\cdot q:\mathcal{A}(N)\in \{p,q\}]<\epsilon(n)P[p←Πn​;q←Πn​;N=p⋅q:A(N)∈{p,q}]<ϵ(n)

Evidence: To this point, best known procedure to always factor has run time O(2nlog(n))O(2^{\sqrt{n}\sqrt{log(n)}})O(2n​log(n)​)

Distribution of prime numbers:

  • We have infinitely many prime
  • Prime Number Theorem π(n)≈nln⁡(n)\pi(n)\approx\frac{n}{\ln(n)}π(n)≈ln(n)n​, that means, 1ln⁡n\frac{1}{\ln n}lnn1​ of all integers are prime.

We want to (guaranteed to) find prime:

π(n)>2n2n\pi(n)>\frac{2^n}{2n}π(n)>2n2n​

e.g.

P[x←{0,1}n:x∈prime]≥2n2n2n=12nP[x\gets \{0,1\}^n:x\in prime]\geq {\frac{2^n}{2n}\over 2^n}=\frac{1}{2n}P[x←{0,1}n:x∈prime]≥2n2n2n​​=2n1​

Theorem:

fmult:{0,1}2n→{0,1}2n,fmult(x1,x2)=x1⋅x2f_{mult}:\{0,1\}^{2n}\to \{0,1\}^{2n},f_{mult}(x_1,x_2)=x_1\cdot x_2fmult​:{0,1}2n→{0,1}2n,fmult​(x1​,x2​)=x1​⋅x2​

Idea: There are enough pairs of primes to make this difficult.

Prove one-way function (under assumptions)

  1. Show ∃p.p.t\exists p.p.t∃p.p.t solves f(x),∀xf(x),\forall xf(x),∀x.
  2. Proof by contradiction.
    • For weak: Provide p(n)p(n)p(n) that we know works.
      • Assume ∃A\exists \mathcal{A}∃A such that P[A inverts]>1−1p(n)P[\mathcal{A}\ \text{inverts}]>1-\frac{1}{p(n)}P[A inverts]>1−p(n)1​
    • For strong: Provide p(n)p(n)p(n) that we know works.

Construct p.p.t B\mathcal{B}B which uses A\mathcal{A}A to solve a problem, which contradicts assumption or known fact.
