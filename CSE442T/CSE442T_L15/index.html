Random Function

F:{0,1}n→{0,1}nF:\{0,1\}^n\to \{0,1\}^nF:{0,1}n→{0,1}n

For each x∈{0,1}nx\in \{0,1\}^nx∈{0,1}n, there are 2n2^n2n possible values for F(x)F(x)F(x).

pick y=F(x)←{0,1}ny=F(x)\gets \{0,1\}^ny=F(x)←{0,1}n independently at random. (nnn bits)

This generates n⋅2nn\cdot 2^nn⋅2n random bits to specify FFF.

Equivalent description of FFF

# initialized empty list L
L=collections.defaultdict(int)
# initialize n bits constant
n=10
def F(x):
    """ simulation of random function
    param:
        x: n bits
    return:
        y: n bits
    """
    if L[x] is not None:
        return L[x]
    else:
        # y is a random n-bit string
        y=random.randbits(n)
        L[x]=y
        return y

However, this is not a good random function since two communicator may not agree on the same FFF.
