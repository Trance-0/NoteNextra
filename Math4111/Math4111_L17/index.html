Definition 3.12

Let XXX be a metric space. We say XXX is complete if every Cauchy sequence in XXX converges.

Theorem 3.11(b) can also be rephrased as:

XXX is a compact metric space   ⟹  \implies⟹ XXX is complete.

Theorem 3.11(c) can also be rephrased as:

Rk\mathbb{R}^kRk is complete.

Note: completeness is a property of the “universe” XXX, not a property of any particular sequence in XXX.

Q\mathbb{Q}Q is not complete. {3,3.1,3.14,3.141,3.1415,… }\{3,3.1,3.14,3.141,3.1415,\dots\}{3,3.1,3.14,3.141,3.1415,…} is a Cauchy sequence in Q\mathbb{Q}Q but it does not converge in Q\mathbb{Q}Q.

Fact: If XXX is complete and EEE is a closed subset of XXX, then EEE is complete.

Definition 3.13

A sequence (sn)(s_n)(sn​) of real numbers is said to be

  • monotone increasing if sn≤sn+1s_n\leq s_{n+1}sn​≤sn+1​ for all nnn.
  • monotone decreasing if sn≥sn+1s_n\geq s_{n+1}sn​≥sn+1​ for all nnn.
  • strictly monotone increasing if sn<sn+1s_n<s_{n+1}sn​<sn+1​ for all nnn.
  • strictly monotone decreasing if sn>sn+1s_n>s_{n+1}sn​>sn+1​ for all nnn.
  • monotone if it is either monotone increasing or monotone decreasing.

Example:

  1. sn=1/ns_n=1/nsn​=1/n is strictly monotone decreasing.
  2. sn=(−1)ns_n=(-1)^nsn​=(−1)n is neither monotone increasing nor monotone decreasing.

Theorem 3.14

Suppose (sn)(s_n)(sn​) is monotonic. Then (sn)(s_n)(sn​) converges   ⟺  \iff⟺ (sn)(s_n)(sn​) is bounded.

Proof:

If (sn)(s_n)(sn​) is monotonic and bounded, then by previous result, (sn)(s_n)(sn​) converges.

If (sn)(s_n)(sn​) is monotonic and converges, then by Theorem 3.2(c), (sn)(s_n)(sn​) is bounded.

QED
