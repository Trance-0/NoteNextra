Weierstrass M-test

Let ∑n=1∞fn(x)\sum_{n=1}^{\infty} f_n(x)∑n=1∞​fn​(x) be a series of functions.

The weierstrass M-test goes as follows:

  1. ∃Mn≥0\exists M_n \geq 0∃Mn​≥0 such that ∀x∈E,∣fn(x)∣≤Mn\forall x\in E, |f_n(x)| \leq M_n∀x∈E,∣fn​(x)∣≤Mn​.
  2. ∑Mn\sum M_n∑Mn​ converges.

Then ∑n=1∞fn(x)\sum_{n=1}^{\infty} f_n(x)∑n=1∞​fn​(x) converges uniformly.

Example:

Ver.0

∀x∈[−1,1)\forall x\in [-1,1)∀x∈[−1,1),

∑n=1∞xnn\sum_{n=1}^{\infty} \frac{x^n}{n}n=1∑∞​nxn​

converges. (point-wise convergence on [−1,1)[-1,1)[−1,1))

∀x∈[−1,1)\forall x\in [-1,1)∀x∈[−1,1),

∣xnn∣≤1n\left| \frac{x^n}{n} \right| \leq \frac{1}{n}​nxn​​≤n1​

Since ∑n=1∞1n\sum_{n=1}^{\infty} \frac{1}{n}∑n=1∞​n1​ diverges, we don’t know if the series converges uniformly or not using the weierstrass M-test.

Ver.1

However, if we consider the series on [−1,1][-1,1][−1,1],

∑n=1∞xnn2\sum_{n=1}^{\infty} \frac{x^n}{n^2}n=1∑∞​n2xn​

converges uniformly. Let Mn=1n2M_n = \frac{1}{n^2}Mn​=n21​. This satisfies the weierstrass M-test. And this series converges uniformly on [−1,1][-1,1][−1,1].

Ver.2

∀x∈[−12,12]\forall x\in [-\frac{1}{2},\frac{1}{2}]∀x∈[−21​,21​],

converges uniformly. Since ∣xnn∣=∣x∣nn≤(1/2)nn≤12n=Mn\left| \frac{x^n}{n} \right|=\frac{|x|^n}{n}\leq \frac{(1/2)^n}{n}\leq \frac{1}{2^n}=M_n​nxn​​=n∣x∣n​≤n(1/2)n​≤2n1​=Mn​, by geometric series test, ∑n=1∞Mn\sum_{n=1}^{\infty} M_n∑n=1∞​Mn​ converges.

M-test still not applicable here.

∑n=1∞xnn\sum_{n=1}^{\infty} \frac{x^n}{n}n=1∑∞​nxn​

converges uniformly on [−12,12][-\frac{1}{2},\frac{1}{2}][−21​,21​].

For a series ∑n=1∞an\sum_{n=1}^{\infty} a_n∑n=1∞​an​, if

  1. ∃Mn\exists M_n∃Mn​ such that ∣an∣≤Mn|a_n|\leq M_n∣an​∣≤Mn​

Proving continuity of a function

If f:E→Yf:E\to Yf:E→Y is continuous at p∈Ep\in Ep∈E, then for any ϵ>0\epsilon>0ϵ>0, there exists δ>0\delta>0δ>0 such that for any x∈Ex\in Ex∈E, if ∣x−p∣<δ|x-p|<\delta∣x−p∣<δ, then ∣f(x)−f(p)∣<ϵ|f(x)-f(p)|<\epsilon∣f(x)−f(p)∣<ϵ.

Example:

Let f(x)=2x+1f(x)=2x+1f(x)=2x+1. For p=1p=1p=1, prove that fff is continuous at ppp.

Let ϵ>0\epsilon>0ϵ>0 be given. Let δ=ϵ2\delta=\frac{\epsilon}{2}δ=2ϵ​. Then for any x∈Rx\in \mathbb{R}x∈R, if ∣x−1∣<δ|x-1|<\delta∣x−1∣<δ, then

∣f(x)−f(1)∣=∣2x+1−3∣=∣2x−2∣=2∣x−1∣<2δ=ϵ.|f(x)-f(1)|=|2x+1-3|=|2x-2|=2|x-1|<2\delta=\epsilon.∣f(x)−f(1)∣=∣2x+1−3∣=∣2x−2∣=2∣x−1∣<2δ=ϵ.

Therefore, fff is continuous at p=1p=1p=1.

You can also use smaller δ\deltaδ and we don’t need to find the “optimal” δ\deltaδ.

Cauchy criterion

In sequences

Def: A sequence {an}\{a_n\}{an​} is Cauchy if for any ϵ>0\epsilon>0ϵ>0, there exists NNN such that for any m,n≥Nm,n\geq Nm,n≥N, ∣am−an∣<ϵ|a_m-a_n|<\epsilon∣am​−an​∣<ϵ.

Theorem: In R\mathbb{R}R, every sequence is Cauchy if and only if it is convergent.

In series

Let sn=∑k=1naks_n=\sum_{k=1}^{n} a_ksn​=∑k=1n​ak​.

Def: A series ∑n=1∞an\sum_{n=1}^{\infty} a_n∑n=1∞​an​ converges if the sequence of partial sums {sn}\{s_n\}{sn​} converges.

∀ϵ>0\forall \epsilon>0∀ϵ>0, there exists NNN such that for any m>n≥Nm>n\geq Nm>n≥N,

∣sm−sn∣=∣∑k=n+1mak∣<ϵ.|s_m-s_n|=\left|\sum_{k=n+1}^{m} a_k\right|<\epsilon.∣sm​−sn​∣=​k=n+1∑m​ak​​<ϵ.

Comparison test

If ∣an∣≤bn|a_n|\leq b_n∣an​∣≤bn​ and ∑n=1∞bn\sum_{n=1}^{\infty} b_n∑n=1∞​bn​ converges, then ∑n=1∞an\sum_{n=1}^{\infty} a_n∑n=1∞​an​ converges.

Proof:

Since ∑n=1∞bn\sum_{n=1}^{\infty} b_n∑n=1∞​bn​ converges, ∀ϵ>0\forall \epsilon>0∀ϵ>0, there exists NNN such that for any m>n≥Nm>n\geq Nm>n≥N,

∣∑k=n+1mbk∣<ϵ.\left|\sum_{k=n+1}^{m} b_k\right|<\epsilon.​k=n+1∑m​bk​​<ϵ.

By triangle inequality,

∣∑k=nmak∣≤∑k=n+1m∣ak∣≤∑k=n+1mbk<ϵ.\left|\sum_{k=n}^{m}a_k\right|\leq \sum_{k=n+1}^{m} |a_k|\leq \sum_{k=n+1}^{m} b_k<\epsilon.​k=n∑m​ak​​≤k=n+1∑m​∣ak​∣≤k=n+1∑m​bk​<ϵ.

Therefore, ∀ϵ>0\forall \epsilon>0∀ϵ>0, there exists NNN such that for any m>n≥Nm>n\geq Nm>n≥N,

∣sm−sn∣=∣∑k=n+1mak∣<ϵ.|s_m-s_n|=\left|\sum_{k=n+1}^{m} a_k\right|<\epsilon.∣sm​−sn​∣=​k=n+1∑m​ak​​<ϵ.

Therefore, {sn}\{s_n\}{sn​} is Cauchy, and ∑n=1∞an\sum_{n=1}^{\infty} a_n∑n=1∞​an​ converges.
