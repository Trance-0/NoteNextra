Continuous functions

Definition 4.5

Let (X,dX)(X,d_X)(X,dX​) and (Y,dY)(Y,d_Y)(Y,dY​) be metric spaces and E⊂XE\subset XE⊂X, p∈Ep\in Ep∈E. Let f:E→Yf:E\to Yf:E→Y. We say that fff is continuous at ppp if ∀ϵ>0\forall \epsilon > 0∀ϵ>0, ∃δ>0\exists \delta > 0∃δ>0 such that f(E∩Bδ(p))⊂Bϵ(f(p))f(E\cap B_\delta(p))\subset B_\epsilon(f(p))f(E∩Bδ​(p))⊂Bϵ​(f(p)).

  • We say that fff is continuous on EEE if fff is continuous at every p∈Ep\in Ep∈E.

Remark: For p∈Ep\in Ep∈E, there are two possibilities.

  • ppp is an isolated point of EEE.
  • ppp is a limit point of EEE.

In the first case, fff is automatically continuous at ppp. (E∩Bδ(p)={p}E\cap B_\delta(p)=\{p\}E∩Bδ​(p)={p} for all δ>0\delta > 0δ>0.)

In the second case, fff is continuous at ppp if and only if lim⁡x→pf(x)=f(p)\lim_{x\to p} f(x) = f(p)limx→p​f(x)=f(p).

Theorem 4.8

A function f:E→Yf:E\to Yf:E→Y is continuous at p∈Ep\in Ep∈E if the pre-image of every open set is open.

Two consequences if f:E→Yf:E\to Yf:E→Y is continuous:

  • Image of compact set is compact. (Implies Extreme Value Theorem)
  • Image of connected set is connected. (Implies Intermediate Value Theorem)
