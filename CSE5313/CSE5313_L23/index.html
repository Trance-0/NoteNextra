Coded Computing

Motivation

Some facts:

  • Moore’s law is saturating.
    • Improving CPU performance is hard.
  • Modern datasets are growing remarkably large.
    • E.g., TikTok, YouTube.
  • Learning tasks are computationally heavy.
    • E.g., training neural networks.

Solution: Distributed Computing for Scalability

  • Offloading computation tasks to multiple computation nodes.
  • Gather and accumulate computation results.
  • E.g., Apache Hadoop, Apache Spark, MapReduce.

General Framework

  • The system involves 1 master node and PPP worker nodes.
  • The master has a dataset DDD and wants f(D)f(D)f(D), where fff is some function.
  • The master partitions D=(D1,⋯ ,DP)D=(D_1,\cdots,D_P)D=(D1​,⋯,DP​), and sends DiD_iDi​ to node iii.
  • Every node iii computes g(Di)g(D_i)g(Di​), where ggg is somem function.
  • Finally, the master collects g(D1),⋯ ,g(DP)g(D_1),\cdots,g(D_P)g(D1​),⋯,g(DP​) and computes f(D)=h(g(D1),⋯ ,g(DP))f(D)=h(g(D_1),\cdots,g(D_P))f(D)=h(g(D1​),⋯,g(DP​)), where hhh is some function.

Challenges

Stragglers

  • Nodes that are significantly slower than the others.

Adversaries

  • Nodes that return errounous results.
    • Computation/communication errors.
    • Adversarial attacks.

Privary

  • Nodes may be curious about the dataset.

Resemblance to communication channel

Suppose f,g=id⁡f,g=\operatorname{id}f,g=id, and let D=(D1,⋯ ,DP)∈FpD=(D_1,\cdots,D_P)\in \mathbb{F}^pD=(D1​,⋯,DP​)∈Fp a message.

  • DiD_iDi​ is a field element
  • F\mathbb{F}F could be R\mathbb{R}R or C\mathbb{C}C, Fq\mathbb{F}^qFq.

Observation: This is a distributed storage system.

  • An erasure - node that does not respond.
  • An error - node that returns errounous results.

Solution:

  • Add redundancy to the message
  • Error-correcting codes.

Coded Distributed Computing

  • The master partitions DDD and encodes it before sending to PPP workers.
  • Workers perform computations on coded data D~\tilde{D}D~ and generate coded results g(D~)g(\tilde{D})g(D~).
  • The master decode the coded results and obtain f(D)=h(g(D~))f(D)=h(g(\tilde{D}))f(D)=h(g(D~)).

Matrix-Vector Multiplication

  • MDS codes.
  • Short-Dot codes.

Matrix-Matrix Multiplication

  • Polynomial codes.
  • MatDot codes.

Polynomial Evaluation

  • Lagrange codes.
  • Application to BLockchain.

Trivial solution - replication

Why no straggler tolerance?

  • We employ an individual worker node iii to compute yi=(ai,…,aiN)⋅(x1,…,xN)Ty_i=(a_i,\ldots,a_{iN})\cdot (x_1,\ldots,x_N)^Tyi​=(ai​,…,aiN​)⋅(x1​,…,xN​)T.

Replicate the computation?

  • Let r+1r+1r+1 nodes compute every yiy_iyi​.

We need P=rM+MP=rM+MP=rM+M worker nodes to tolerate rrr erasures and ⌊r2⌋\lfloor \frac{r}{2}\rfloor⌊2r​⌋ adversaries.

Use of MDS codes

Let 2∣M2|M2∣M and P=3P=3P=3.

Let A1,A2A_1,A_2A1​,A2​ be submatrices of AAA such that A=[A1⊤∣A2⊤]⊤A=[A_1^\top|A_2^\top]^\topA=[A1⊤​∣A2⊤​]⊤.

  • Worker node 1 conputes A1⋅xA_1\cdot xA1​⋅x.
  • Worker node 2 conputes A2⋅xA_2\cdot xA2​⋅x.
  • Worker node 3 conputes (A1+A2)⋅x(A_1+A_2)\cdot x(A1​+A2​)⋅x.

Observation: the results can be obtained from any two worker nodes.

Let G∈FM×PG\in \mathbb{F}^{M\times P}G∈FM×P be the generator matrix of an (P,M)(P,M)(P,M) MDS code.

The master node computes F=G⊤A∈FP×NF=G^\top A\in \mathbb{F}^{P\times N}F=G⊤A∈FP×N.

Every worker node iii computers Fi⋅xF_i\cdot xFi​⋅x.

Notice that Fx=G⊤A⋅x=G⊤yFx=G^\top A\cdot x=G^\top yFx=G⊤A⋅x=G⊤y is the codeword of yyy.

  • The MDS property allows decoding from any MMM yiy_iyi​‘s
  • This scheme tolerates P−MP-MP−M erasures, and the recovery threshold K=MK=MK=M.
  • We need P=r+MP=r+MP=r+M worker nodes to tolerate rrr stragglers or r2\frac{r}{2}2r​ adversaries.
    • With replication, we need P=rM+MP=rM+MP=rM+M worker nodes.

Potential improvements for MDS codes

  • The matrix AAA is usually a (trained) model, and xxx is the data (feature vector).
  • xxx is transmitted frequently, while the row of AAA (or G⊤AG^\top AG⊤A) is communicated in advance.
  • Every worker needs to receive the entire xxx and compute the dot product.
  • Communication-heavy
  • Can we design a scheme that allows every node only receive only a part of xxx?

Short-Dot codes

link to paper 

We want to create a matrix F∈FP×MF\in \mathbb{F}^{P\times M}F∈FP×M from AAA such that:

  • Every node computes Fi⋅xF_i\cdot xFi​⋅x.
  • Every KKK rows linearly span the row space of AAA.
  • Each row of FFF contains at most sss non-zero entries.

In the MDS method, F=G⊤AF=G^\top AF=G⊤A.

  • The recovery threshold K=MK=MK=M.
  • Every worker node needs to receive s=Ns=Ns=N symbols (the entire x)

No free lunch

Can we trade the recovery threshold KKK for a smaller sss?

  • Every worker node receives less than NNN symbols.
  • The master will need more than MMM responses to recover the computation result.

Construction of Short-Dot codes

Choose a super-regular matrix B∈FP×KB\in \mathbb{F}^{P\times K}B∈FP×K, where PPP is the number of worker nodes.

  • A matrix is supper-regular if every square submatrix is invertible.
  • Lagrange/Cauchy matrix is super-regular (next lecture).

Create matrix A~\tilde{A}A~ by stacking some Z∈F(K−M)×NZ\in \mathbb{F}^{(K-M)\times N}Z∈F(K−M)×N below matrix AAA.

Let F=B⋅A~∈FP×NF=B\cdot \tilde{A}\in \mathbb{F}^{P\times N}F=B⋅A~∈FP×N.

Short-Dot: create matrix F∈FP×NF\in \mathbb{F}^{P\times N}F∈FP×N such that:

  • Every KKK rows linearly span the row space of AAA.
  • Each row of FFF contains at most s=P−K+MPs=\frac{P-K+M}{P}s=PP−K+M​. NNN non-zero entries (sparse).

Recovery of Short-Dot codes

Claim: Every KKK rows of FFF linearly span the row space of AAA.

Proof

Since BBB is supper-regular, it is also MDS, i.e., every K×KK\times KK×K submatrix of BBB is invertible.

Hence, every row of AAA can be represented as a linear combination of any KKK rows of FFF.

That is, for every X⊆[P],∣X∣=K\mathcal{X}\subseteq[P],|\mathcal{X}|=KX⊆[P],∣X∣=K, we can have A~=(BX)−1FX\tilde{A}=(B^{\mathcal{X}})^{-1}F^{\mathcal{X}}A~=(BX)−1FX.
