Finite Fields

Why finite fields?

Most information systems are discrete.

  • Use bits, byte etc.

Use bits/bytes to represent real numbers.

  • Problems of overflow, accuracy, etc.

We wish to build “good” codes C⊂Fn\mathcal{C} \subset \mathbb{F}^nC⊂Fn:

  • Large kn\frac{k}{n}nk​
  • Lage dH(C)  ⟹  d_H(\mathcal{C})\impliesdH​(C)⟹ error detection/correction, erasure correction.

Idea: Use linear algebraic operations to encode/decode.

  • F=FqF=\mathbb{F}_qF=Fq​, a finite field with qqq elements.

Finite fields

Fields and field axioms

A field is a set F\mathbb{F}F with two operations +++ and ⋅\cdot⋅ that satisfy the following axioms:

  • Associativity: (a+b)+c=a+(b+c)(a+b)+c = a+(b+c)(a+b)+c=a+(b+c) and (a⋅b)⋅c=a⋅(b⋅c)(a\cdot b)\cdot c = a\cdot (b\cdot c)(a⋅b)⋅c=a⋅(b⋅c)
  • Commutativity: a+b=b+aa+b = b+aa+b=b+a and a⋅b=b⋅aa\cdot b = b\cdot aa⋅b=b⋅a
  • Distributivity: a⋅(b+c)=a⋅b+a⋅ca\cdot (b+c) = a\cdot b + a\cdot ca⋅(b+c)=a⋅b+a⋅c
  • Existence of Identity elements: a+0=aa+0 = aa+0=a and a⋅1=aa\cdot 1 = aa⋅1=a
  • Existence of Inverse elements: a+(−a)=0a+(-a) = 0a+(−a)=0 and a⋅a−1=1a\cdot a^{-1} = 1a⋅a−1=1

Every set of elements which satisfies these axioms is a field.

We can “do algebra” over it (matrices, vector spaces, etc.).

Are there finite sets which satisfy the field axioms?

What are the possible sizes of such sets?

Background – Basic number theory

  • For a,b∈Na, b \in \mathbb{N}a,b∈N,
    • Greatest Common Denominator: gcd⁡(a,b)=\gcd(a, b) =gcd(a,b)= the largest integer mmm such that m∣am|am∣a and m∣bm|bm∣b.
    • Lowest Common Multiplier: lcm⁡(a,b)=\operatorname{lcm}(a, b) =lcm(a,b)= the smallest integer mmm such that a∣ma|ma∣m and b∣mb|mb∣m.
  • a,ba, ba,b are coprime if gcd⁡(a,b)=1\gcd(a, b) = 1gcd(a,b)=1.
  • Fact: (Euclid’s lemma) Say a≥ba \geq ba≥b,
    • There exists a quotient q≥0q \geq 0q≥0 and a remainder 0≤r<b0 \leq r < b0≤r<b such that a=bq+ra = bq + ra=bq+r.
  • Theorem (Euclid): If gcd⁡(a,b)=1\gcd(a, b) = 1gcd(a,b)=1 then there exist m,n∈Zm, n \in \mathbb{Z}m,n∈Z such that am+bn=1am + bn = 1am+bn=1.
    • Proof by repeated application of Euclid’s lemma.
    • Example:
      • If a=3,b=8a = 3, b = 8a=3,b=8,
      • then m=−5,n=2m = -5, n = 2m=−5,n=2,
      • satisfy 3⋅−5+8⋅2=13 \cdot -5 + 8 \cdot 2 = 13⋅−5+8⋅2=1.

Modular arithmetic

Defined a set with addition ⊕\oplus⊕ and multiplication ⊙\odot⊙ that satisfy the field axioms.

Zp\mathbb{Z}_pZp​ is a field if ppp is a prime number.

  • Addition and multiplication are defined modulo ppp.

  • a⊕b=(a+b)mod  pa \oplus b = (a+b) \mod pa⊕b=(a+b)modp

  • a⊙b=(a⋅b)mod  pa \odot b = (a\cdot b) \mod pa⊙b=(a⋅b)modp

  • 000 is the additive identity.

  • 111 is the multiplicative identity.

  • aaa has an additive inverse p−ap-ap−a.

  • aaa has a multiplicative inverse a−1a^{-1}a−1 such that a⊙a−1=1a \odot a^{-1} = 1a⊙a−1=1.

Proof for existence of multiplicative inverse for a∈Zp∖{0}a\in \mathbb{Z}_p\setminus \{0\}a∈Zp​∖{0}:

Proof

Since ppp is prime, gcd⁡(a,p)=1\gcd(a, p) = 1gcd(a,p)=1.

By euclid’s theorem, there exist m,n∈Zm, n \in \mathbb{Z}m,n∈Z such that am+pn=1am + pn = 1am+pn=1.

Take mod ppp on both sides:

amod  p⊙mmod  p≡1mod  pa_{\mod p}\odot m_{\mod p} \equiv 1_{\mod p}amodp​⊙mmodp​≡1modp​

Thus, mmod  pm_{\mod p}mmodp​ is the multiplicative inverse of amod  pa_{\mod p}amodp​.

Irreducible polynomials

A polynomial p(x)p(x)p(x) is irreducible if it cannot be factored into two non-constant polynomials.

If gcd⁡(a(x),b(x))=1\gcd(a(x),b(x))=1gcd(a(x),b(x))=1, then there exist m(x),n(x)∈Zp[x]m(x),n(x)\in \mathbb{Z}_p[x]m(x),n(x)∈Zp​[x] such that a(x)m(x)⊕b(x)n(x)=1a(x)m(x)\oplus b(x)n(x)=1a(x)m(x)⊕b(x)n(x)=1.

Proved similar to euclid’s theorem.

If a polynomial p(x)p(x)p(x) has a root, say rrr, then p(x)=(x−r)q(x)p(x) = (x-r)q(x)p(x)=(x−r)q(x) for some q(x)∈Zp[x]q(x)\in \mathbb{Z}_p[x]q(x)∈Zp​[x].

p(x)=x2⊕1p(x) = x^2 \oplus 1p(x)=x2⊕1

Extension fields

Let ppp be a prime number. then (Zp[x],⊕,⊙)(\mathbb{Z}_p[x], \oplus, \odot)(Zp​[x],⊕,⊙) is a field.

Fix a polynomial f(x)∈Zp[x]f(x)\in \mathbb{Z}_p[x]f(x)∈Zp​[x] of degree ttt.

Define a set

Elements: polynomials of degree at most t−1t-1t−1 in Zp[x]\mathbb{Z}_p[x]Zp​[x]. (finite set, size is ptp^tpt.)

Define addition:

a(x)⊕fb(x)=(a(x)⊕b(x))mod  f(x)a(x) \oplus_f b(x) = (a(x) \oplus b(x)) \mod f(x)a(x)⊕f​b(x)=(a(x)⊕b(x))modf(x)

Define multiplication:

a(x)⊙fb(x)=(a(x)⊙b(x))mod  f(x)a(x) \odot_f b(x) = (a(x) \odot b(x)) \mod f(x)a(x)⊙f​b(x)=(a(x)⊙b(x))modf(x)

Denote this set as Zp[x]mod  f(x)\mathbb{Z}_p[x] \mod f(x)Zp​[x]modf(x).

This is not a field because it does not have a multiplicative inverse for every element.

Proof

We prove by contradiction.

Suppose there exists a polynomial g(x)∈Zp[x]mod  f(x)g(x)\in \mathbb{Z}_p[x] \mod f(x)g(x)∈Zp​[x]modf(x) such that a(x)⊙fg(x)=1a(x) \odot_f g(x) = 1a(x)⊙f​g(x)=1.

Let p=2,f(x)=x2⊕1p=2,f(x)=x^2\oplus 1p=2,f(x)=x2⊕1.

The polynomials in Z2[x]mod  f(x)\mathbb{Z}_2[x] \mod f(x)Z2​[x]modf(x) are {0,1,x,x⊕1}\{0, 1, x, x\oplus 1\}{0,1,x,x⊕1}.

Consider the modular inverse of (x⊕1)(x\oplus 1)(x⊕1).

  • 0⊙f(x⊕1)=00\odot_f (x\oplus 1) = 00⊙f​(x⊕1)=0
  • 1⊙f(x⊕1)=x⊕11\odot_f (x\oplus 1) = x\oplus 11⊙f​(x⊕1)=x⊕1
  • x⊙f(x⊕1)=(x2⊕x)mod  (x2⊕1)=x⊕1x\odot_f (x\oplus 1) = (x^2\oplus x)\mod (x^2\oplus 1) = x\oplus 1x⊙f​(x⊕1)=(x2⊕x)mod(x2⊕1)=x⊕1
  • (x⊕1)⊙f(x⊕1)=(x2⊕1)mod  (x2⊕1)=0(x\oplus 1)\odot_f (x\oplus 1) = (x^2\oplus 1)\mod (x^2\oplus 1) = 0(x⊕1)⊙f​(x⊕1)=(x2⊕1)mod(x2⊕1)=0

Theorem: If f(x)f(x)f(x) is irreducible over Zp\mathbb{Z}_pZp​, then Zp[x]mod  f(x)\mathbb{Z}_p[x] \mod f(x)Zp​[x]modf(x) is a field.

Proof

Let a(x)∈Zp[x]mod  f(x)a(x)\in \mathbb{Z}_p[x] \mod f(x)a(x)∈Zp​[x]modf(x), a(x)≠0a(x)\neq 0a(x)=0.

Take mod f(x)f(x)f(x) on both sides:

a(x)m(x)mod  f(x)=1mod  f(x)a(x)m(x) \mod f(x) = 1 \mod f(x)a(x)m(x)modf(x)=1modf(x)

Thus, m(x)mod  f(x)m(x) \mod f(x)m(x)modf(x) is the multiplicative inverse of a(x)mod  f(x)a(x) \mod f(x)a(x)modf(x).

So a(x)−1=m(x)mod  f(x)a(x)^{-1} = m(x) \mod f(x)a(x)−1=m(x)modf(x).

Uniqueness of the finite field

Theorems:

  • As long as it is irreducible, the choice of f(x)f(x)f(x) does not matter.
    • If f1(x),f2(x)f_1(x), f_2(x)f1​(x),f2​(x) are irreducible of the same degree, then Zp[x]mod  f1(x)≅Zp[x]mod  f2(x)\mathbb{Z}_p[x] \mod f_1(x) \cong \mathbb{Z}_p[x] \mod f_2(x)Zp​[x]modf1​(x)≅Zp​[x]modf2​(x).
  • Over every Zp\mathbb{Z}_pZp​ (𝑝 prime), there exists an irreducible polynomial of every degree.
  • All finite fields of the same size are isomorphic.
  • All finite fields are of size pdp^dpd for prime ppp and integer ddd.

Corollary: This is effectively the only way to construct finite fields!
