Algebra over finite fields

N\mathbb{N}N is the set of natural numbers.

by adding additive inverse, we can define the set of integers Z\mathbb{Z}Z.

by adding multiplicative inverse, we can define the set of rational numbers Q\mathbb{Q}Q.

by adding limits, we can define the set of real numbers R\mathbb{R}R.

by adding polynomial roots, we can define the set of complex numbers C\mathbb{C}C.

Another two extensions H\mathbb{H}H for quaternions and O\mathbb{O}O for octonions.

Few theorems about extension of fields

  • As long as it is irreducible, the choice of f(x)f(x)f(x) does not matter.
    • If f1(x),f2(x)f_1(x), f_2(x)f1​(x),f2​(x) are irreducible of the same degree, then Zp[x]mod  f1(x)≅Zp[x]mod  f2(x)\mathbb{Z}_p[x] \mod f_1(x) \cong \mathbb{Z}_p[x] \mod f_2(x)Zp​[x]modf1​(x)≅Zp​[x]modf2​(x).
  • Over every Zp\mathbb{Z}_pZp​ (ppp prime), there exists an irreducible polynomial of every degree.
  • All finite fields of the same size are isomorphic.
  • All finite fields are of size pdp^dpd for prime ppp and integer ddd.

Corollary: This is effectively the only way to construct finite fields.

Common notations

Fq\mathbb{F}_qFq​ is the finite field with qqq elements. where qqq is a prime power.

Fqt\mathbb{F}_q^tFqt​ is a vector field of dimension ttt over Fq\mathbb{F}_qFq​. (no notion of multiplication)

Fqt\mathbb{F}_{q^t}Fqt​ is the finite field with qtq^tqt elements. (as extension field of Fq\mathbb{F}_qFq​)

Fqt⇒Fqt\mathbb{F}_{q^t}\Rightarrow \mathbb{F}_q^tFqt​⇒Fqt​. Every extension field of Fq\mathbb{F}_qFq​ is a vector field over Fq\mathbb{F}_qFq​.

Fqt⇏Fqt\mathbb{F}_q^t\nRightarrow \mathbb{F}_{q^t}Fqt​⇏Fqt​. Additional structure is required.

Important

From now on, we will use +,⋅+,\cdot+,⋅ to denote the modulo operations ⊕,⊙\oplus,\odot⊕,⊙.

Groups

A group is a set GGG with an operation ⋅\cdot⋅ that satisfies the following axioms:

  1. Closure: ∀a,b∈G,a⋅b∈G\forall a,b\in G, a\cdot b\in G∀a,b∈G,a⋅b∈G.
  2. Associativity: ∀a,b,c∈G,(a⋅b)⋅c=a⋅(b⋅c)\forall a,b,c\in G, (a\cdot b)\cdot c=a\cdot (b\cdot c)∀a,b,c∈G,(a⋅b)⋅c=a⋅(b⋅c).
  3. Identity: ∃e∈G,∀a∈G,a⋅e=e⋅a=a\exists e\in G, \forall a\in G, a\cdot e=e\cdot a=a∃e∈G,∀a∈G,a⋅e=e⋅a=a.
  4. Inverses: ∀a∈G,∃a−1∈G,a⋅a−1=a−1⋅a=e\forall a\in G, \exists a^{-1}\in G, a\cdot a^{-1}=a^{-1}\cdot a=e∀a∈G,∃a−1∈G,a⋅a−1=a−1⋅a=e.
Important
  1. Operator ⋅\cdot⋅ is not necessarily commutative. (if so then it’s called an abelian group)
  2. May not be finite/infinite.
  3. May represented in power notation an=an−1⋅aa^n=a^{n-1}\cdot aan=an−1⋅a.

Order of element in group

Let (G,⋅)(G,\cdot)(G,⋅) be a finite group.

Then there exists k∈Nk\in\mathbb{N}k∈N such that ak=ea^k=eak=e.

Proof

Consider the sequence a,a2,a3,⋯a,a^2,a^3,\cdotsa,a2,a3,⋯.

Since GGG is finite, there exists i,j∈Ni,j\in\mathbb{N}i,j∈N such that ai=aja^i=a^jai=aj.

Then ai=aj  ⟺  ai−j=ea^i=a^j\iff a^{i-j}=eai=aj⟺ai−j=e.

So there exists k=i−j∈Nk=i-j\in\mathbb{N}k=i−j∈N such that ak=ea^k=eak=e.

The order of an element a∈Ga\in Ga∈G (denoted as O(a)\mathcal{O}(a)O(a)) is the smallest positive integer nnn such that an=ea^n=ean=e.
