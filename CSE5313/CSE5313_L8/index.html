Boundary of linear codes

Natural questions:

  • Can we extend the table of linear codes infinitely?
  • What set of configuration (n,k,d)q(n,k,d)_q(n,k,d)q​ are impossible?
  • What set of configuration (n,k,d)q(n,k,d)_q(n,k,d)q​ are possible, even if we don’t know how to construct them?

Boundary I: Singleton bound

Singleton is a name for the person who discovered this bound.

Theorem: For any linear code C⊆Fqn\mathcal{C}\subseteq \mathbb{F}^n_qC⊆Fqn​, we have d≤n−k+1d\leq n-k+1d≤n−k+1.

Proof

Idea: Using the Pigeonhole principle.

Assume an code [n,k,d]q[n,k,d]_q[n,k,d]q​ exists.

Pigeons: All qkq^kqk possible code word of C\mathcal{C}C.

Holes: All qℓq^\ellqℓ values of the first ℓ\ellℓ entries of a codeword (for some ℓ<n\ell<nℓ<n).

If qℓ<qkq^\ell<q^kqℓ<qk, then by Pigeonhole principle, there exists two codewords in C\mathcal{C}C that agrees on the first ℓ\ellℓ entries.

So d≤n−ℓd\leq n-\elld≤n−ℓ.

So the largest ℓ\ellℓ value for which this arguments works is n−k+1n-k+1n−k+1.

Definition of Maximum Distance Separable (MDS) code

A code C=[n,k,d]q\mathcal{C} = [n,k,d]_qC=[n,k,d]q​ with d=n−k+1d = n - k + 1d=n−k+1 is called a Maximum Distance Separable (MDS) code.

Examples for singleton bound

Fn\mathbb{F}^nFn: any n,k=n,d=1n, k = n, d = 1n,k=n,d=1.

  • Attains with equality!

Parity: any n,k=n−1,d=2n, k = n - 1, d = 2n,k=n−1,d=2.

  • Attains with equality!

Hamming: n=2m−1,k=2m−m−1,d=3n = 2^m - 1, k = 2^m - m - 1, d = 3n=2m−1,k=2m−m−1,d=3.

n−k+1=m+1>3n - k + 1 = m + 1 > 3n−k+1=m+1>3.

This creates some trade-off between kkk and ddd.
