Learn in Latent Space: Dreamer

Learning embedding of images & dynamics model (jointly)

Representation model: pθ(st∣st−1,at−1,ot)p_\theta(s_t|s_{t-1}, a_{t-1}, o_t)pθ​(st​∣st−1​,at−1​,ot​)

Observation model: qθ(ot∣st)q_\theta(o_t|s_t)qθ​(ot​∣st​)

Reward model: qθ(rt∣st)q_\theta(r_t|s_t)qθ​(rt​∣st​)

Transition model: qθ(st∣st−1,at−1)q_\theta(s_t| s_{t-1}, a_{t-1})qθ​(st​∣st−1​,at−1​).

Variational evidence lower bound (ELBO) objective:

JREC≐Ep(∑t(JOt+JRt+JDt))\mathcal{J}_{REC}\doteq \mathbb{E}_{p}\left(\sum_t(\mathcal{J}_O^t+\mathcal{J}_R^t+\mathcal{J}_D^t)\right)JREC​≐Ep​(t∑​(JOt​+JRt​+JDt​))

where

JOt≐ln⁡q(ot∣st)\mathcal{J}_O^t\doteq \ln q(o_t|s_t)JOt​≐lnq(ot​∣st​) JRt≐ln⁡q(rt∣st)\mathcal{J}_R^t\doteq \ln q(r_t|s_t)JRt​≐lnq(rt​∣st​) JDt≐−βKL⁡(p(st∣st−1,at−1,ot)∣∣q(st∣st−1,at−1))\mathcal{J}_D^t\doteq -\beta \operatorname{KL}(p(s_t|s_{t-1}, a_{t-1}, o_t)||q(s_t|s_{t-1}, a_{t-1}))JDt​≐−βKL(p(st​∣st−1​,at−1​,ot​)∣∣q(st​∣st−1​,at−1​))

Learn in Latent Space

  • Pros
    • Learn visual skill efficiently (using relative simple networks)
  • Cons
    • Using autoencoder might not recover the right representation
    • Not necessarily suitable for model-based methods
    • Embedding is often not a good state representation without using history observations
