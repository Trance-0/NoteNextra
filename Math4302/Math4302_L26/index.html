Fermat’s and Euler’s Theorems

Recall from last lecture, we consider Zp\mathbb{Z}_pZp​ and Zp∗\mathbb{Z}_p^*Zp∗​ denote the group of units in Zp\mathbb{Z}_pZp​ with multiplication.

Zp∗={1,2,⋯ ,p−1},∣Zp∗∣=p−1\mathbb{Z}_p^* = \{1,2,\cdots,p-1\}, \quad |\mathbb{Z}_p^*| = p-1Zp∗​={1,2,⋯,p−1},∣Zp∗​∣=p−1

Let [a]∈Zp∗[a]\in \mathbb{Z}_p^*[a]∈Zp∗​, then [a]p−1=[1][a]^{p-1}=[1][a]p−1=[1], this implies that ap−1mod  p=1a^{p-1}\mod p=1ap−1modp=1.

Now if m∈Zm\in \mathbb{Z}m∈Z and a=a=a= remainder of mmm by ppp, [a]∈Zp[a]\in \mathbb{Z}_p[a]∈Zp​, implies m≡amod  pm\equiv a\mod pm≡amodp.

Then mp−1≡ap−1mod  pm^{p-1}\equiv a^{p-1}\mod pmp−1≡ap−1modp.

So

Fermat’s little theorem

If ppp is not a divisor of mmm, then mp−1≡1mod  pm^{p-1}\equiv 1\mod pmp−1≡1modp.

Corollary of Fermat’s little theorem

If m∈Zm\in \mathbb{Z}m∈Z, then mp≡mmod  pm^p\equiv m\mod pmp≡mmodp.

Proof

If p∣mp|mp∣m, then mp−1≡0≡mmod  pm^{p-1}\equiv 0\equiv m\mod pmp−1≡0≡mmodp.

If p∤mp\not|mp∣m, then by Fermat’s little theorem, mp−1≡1≡mmod  pm^{p-1}\equiv 1\equiv m\mod pmp−1≡1≡mmodp, so mp≡mmod  pm^p\equiv m\mod pmp≡mmodp.

210≡(−6)2mod  19≡36mod  19≡17mod  192^10\equiv (-6)^2\mod 19\equiv 36\mod 19\equiv 17\mod 19210≡(−6)2mod19≡36mod19≡17mod19

Euler’s totient function

Consider Z6\mathbb{Z}_6Z6​, by definition for the group of units, Z6∗={1,5}\mathbb{Z}_6^*=\{1,5\}Z6∗​={1,5}.

ϕ(n)=∣Zn∗∣=∣{1≤x≤n:gcd(x,n)=1}∣\phi(n)=|\mathbb{Z}_n^*|=|\{1\leq x\leq n:gcd(x,n)=1\}|ϕ(n)=∣Zn∗​∣=∣{1≤x≤n:gcd(x,n)=1}∣

Example

ϕ(8)=∣{1,3,5,7}∣=4\phi(8)=|\{1,3,5,7\}|=4ϕ(8)=∣{1,3,5,7}∣=4

If [a]∈Zn∗[a]\in \mathbb{Z}_n^*[a]∈Zn∗​, then [a]ϕ(n)=[1][a]^{\phi(n)}=[1][a]ϕ(n)=[1]. So aϕ(n)≡1mod  na^{\phi(n)}\equiv 1\mod naϕ(n)≡1modn.

Euler’s Theorem

If m∈Zm\in \mathbb{Z}m∈Z, and gcd(m,n)=1gcd(m,n)=1gcd(m,n)=1, then mϕ(n)≡1mod  nm^{\phi(n)}\equiv 1\mod nmϕ(n)≡1modn.

Proof

If aaa is the remainder of mmm by nnn, then m≡amod  nm\equiv a\mod nm≡amodn, and gcd⁡(a,n)=1\operatorname{gcd}(a,n)=1gcd(a,n)=1, so mϕ(n)≡aϕ(n)≡1mod  nm^{\phi(n)}\equiv a^{\phi(n)}\equiv 1\mod nmϕ(n)≡aϕ(n)≡1modn.

Applications on solving modular equations

Solving equations of the form ax≡bmod  nax\equiv b\mod nax≡bmodn.

Not always have solution, 2x≡1mod  42x\equiv 1\mod 42x≡1mod4 has no solution since 111 is odd.

Solution for 2x≡1mod  32x\equiv 1\mod 32x≡1mod3

  • x≡0  ⟹  2x≡0mod  3x\equiv 0\implies 2x\equiv 0\mod 3x≡0⟹2x≡0mod3
  • x≡1  ⟹  2x≡2mod  3x\equiv 1\implies 2x\equiv 2\mod 3x≡1⟹2x≡2mod3
  • x≡2  ⟹  2x≡1mod  3x\equiv 2\implies 2x\equiv 1\mod 3x≡2⟹2x≡1mod3

So solution for 2x≡1mod  32x\equiv 1\mod 32x≡1mod3 is {3k+2∣k∈Z}\{3k+2|k\in \mathbb{Z}\}{3k+2∣k∈Z}.

Theorem for existence of solution of modular equations

ax≡bmod  nax\equiv b\mod nax≡bmodn has a solution if and only if gcd⁡(a,n)∣b\operatorname{gcd}(a,n)|bgcd(a,n)∣b and in that case the equation has ddd solutions in Zn\mathbb{Z}_nZn​.
