Dihedral groups

The dihedral group DnD_nDn​ is the group of symmetries of a regular nnn-gon.

(Permutation that sends adjacent vertices to adjacent vertices)

Dn<SnD_n<S_nDn​<Sn​

∣Sn∣=n!,∣Dn∣=2n|S_n|=n!, |D_n|=2n∣Sn​∣=n!,∣Dn​∣=2n

We can classify dihedral groups as follows:

ρ∈Dn\rho \in D_nρ∈Dn​ as the rotation of a regular nnn-gon by 2πn\frac{2\pi}{n}n2π​.

ϕ∈Dn\phi\in D_nϕ∈Dn​ as a reflection of a regular nnn-gon with respect to xxx-axis.

We can enumerate the elements of DnD_nDn​ as follows:

Dn=⟨ϕ,ρ⟩={e,ρ,ρ2,⋯ ,ρn−1,ϕ,ϕρ,ϕρ2,⋯ ,ϕρn−1}D_n=\langle \phi,\rho\rangle=\{e,\rho,\rho^2,\cdots,\rho^{n-1},\phi,\phi\rho,\phi\rho^2,\cdots,\phi\rho^{n-1}\}Dn​=⟨ϕ,ρ⟩={e,ρ,ρ2,⋯,ρn−1,ϕ,ϕρ,ϕρ2,⋯,ϕρn−1}

We claim these elements are all distinct.

Proof

Consider the first half, clearly ρi≠ρj\rho_i\neq \rho_jρi​=ρj​ if 0≤i<j≤n−10\leq i<j\leq n-10≤i<j≤n−1.

Also ϕρi≠ϕρj\phi\rho_i\neq \phi\rho_jϕρi​=ϕρj​ if 0≤i<j≤n−10\leq i<j\leq n-10≤i<j≤n−1. otherwise ρi=ρj\rho_i=\rho_jρi​=ρj​

Also ρi≠ρjϕ\rho^i\neq \rho^j\phiρi=ρjϕ where 0≤i,j≤n−10\leq i,j\leq n-10≤i,j≤n−1.

Otherwise ρi−j=ϕ\rho^{i-j}=\phiρi−j=ϕ, but reflection (with some point fixed) cannot be any rotation (no points are fixed).

In DnD_nDn​, ϕρ=ρn−1ϕ\phi\rho=\rho^{n-1}\phiϕρ=ρn−1ϕ, more generally, ϕρi=ρn−iϕ\phi\rho^i=\rho^{n-i}\phiϕρi=ρn−iϕ for any i∈Zi\in\mathbb{Z}i∈Z.

Group homomorphism

Definition for group homomorphism

Let G,G′G,G'G,G′ be groups.

ϕ:G→G′\phi:G\to G'ϕ:G→G′ is called a group homomorphism if ϕ(g1g2)=ϕ(g1)ϕ(g2)\phi(g_1g_2)=\phi(g_1)\phi(g_2)ϕ(g1​g2​)=ϕ(g1​)ϕ(g2​) for all g1,g2∈Gg_1,g_2\in Gg1​,g2​∈G (Note that ϕ\phiϕ may not be bijective).

This is a weaker condition than isomorphism.

Example

GL(2,R)={A∈M2×2(R)∣det(A)≠0}GL(2,\mathbb{R})=\{A\in M_{2\times 2}(\mathbb{R})|det(A)\neq 0\}GL(2,R)={A∈M2×2​(R)∣det(A)=0}

Then ϕ:GL(2,R)→(R−{0},⋅)\phi:GL(2,\mathbb{R})\to (\mathbb{R}-\{0\},\cdot)ϕ:GL(2,R)→(R−{0},⋅) where ϕ(A)=det⁡(A)\phi(A)=\det(A)ϕ(A)=det(A) is a group homomorphism, since det⁡(AB)=det⁡(A)det⁡(B)\det(AB)=\det(A)\det(B)det(AB)=det(A)det(B).

This is not one-to-one but onto, therefore not an isomorphism.

(Zn,+)(\mathbb{Z}_n,+)(Zn​,+) and DnD_nDn​ has homomorphism (Zn,+)→Dn(\mathbb{Z}_n,+)\to D_n(Zn​,+)→Dn​ where ϕ(k)=ρk\phi(k)=\rho^kϕ(k)=ρk

ϕ(i+j)=ρi+jmod  n=ρiρj=ϕ(i)+ϕ(j)\phi(i+j)=\rho^{i+j\mod n}=\rho^i\rho^j=\phi(i)+\phi(j)ϕ(i+j)=ρi+jmodn=ρiρj=ϕ(i)+ϕ(j).

This is not onto but one-to-one, therefore not an isomorphism.

Let G,G′G,G'G,G′ be two groups, let eee be the identity of GGG and let e′e'e′ be the identity of G′G'G′.

Let ϕ:G→G′\phi:G\to G'ϕ:G→G′, ϕ(a)=e′\phi(a)=e'ϕ(a)=e′ for all a∈Ga\in Ga∈G.

ϕ(ab)=ϕ(a)ϕ(b)=e′e′=e′\phi(ab)=\phi(a)\phi(b)=e'e'=e'ϕ(ab)=ϕ(a)ϕ(b)=e′e′=e′

This is generally not onto and not one-to-one, therefore not an isomorphism.

Corollary for group homomorphism

Let G,G′G,G'G,G′ be groups and ϕ:G→G′\phi:G\to G'ϕ:G→G′ be a group homomorphism. eee is the identity of GGG and e′e'e′ is the identity of G′G'G′.

  1. ϕ(e)=e′\phi(e)=e'ϕ(e)=e′
  2. ϕ(a−1)=(ϕ(a))−1\phi(a^{-1})=(\phi(a))^{-1}ϕ(a−1)=(ϕ(a))−1 for all a∈Ga\in Ga∈G
  3. If H≤GH\leq GH≤G, then ϕ(H)≤G′\phi(H)\leq G'ϕ(H)≤G′, where ϕ(H)={ϕ(a)∣a∈H}\phi(H)=\{\phi(a)|a\in H\}ϕ(H)={ϕ(a)∣a∈H}.
  4. If K≤G′K\leq G'K≤G′ then ϕ−1(K)≤G\phi^{-1}(K)\leq Gϕ−1(K)≤G, where ϕ−1(K)={a∈G∣ϕ(a)∈K}\phi^{-1}(K)=\{a\in G|\phi(a)\in K\}ϕ−1(K)={a∈G∣ϕ(a)∈K}.

Proof

(1) ϕ(e)=e′\phi(e)=e'ϕ(e)=e′

Consider ϕ(ee)=ϕ(e)ϕ(e)\phi(ee)=\phi(e)\phi(e)ϕ(ee)=ϕ(e)ϕ(e), therefore ϕ(e)=e′\phi(e)=e'ϕ(e)=e′ by cancellation on the left.

(2) ϕ(a−1)=(ϕ(a))−1\phi(a^{-1})=(\phi(a))^{-1}ϕ(a−1)=(ϕ(a))−1

Consider ϕ(a−1a)=ϕ(a−1)ϕ(a)=ϕ(e)\phi(a^{-1}a)=\phi(a^{-1})\phi(a)=\phi(e)ϕ(a−1a)=ϕ(a−1)ϕ(a)=ϕ(e), therefore ϕ(a−1)\phi(a^{-1})ϕ(a−1) is the inverse of ϕ(a)\phi(a)ϕ(a) in G′G'G′.

(3) If H≤GH\leq GH≤G, then ϕ(H)≤G′\phi(H)\leq G'ϕ(H)≤G′, where ϕ(H)={ϕ(a)∣a∈H}\phi(H)=\{\phi(a)|a\in H\}ϕ(H)={ϕ(a)∣a∈H}.

  • e∈He\in He∈H implies that e′=ϕ(e)∈ϕ(H)e'=\phi(e)\in\phi(H)e′=ϕ(e)∈ϕ(H).
  • If x∈ϕ(H)x\in \phi(H)x∈ϕ(H), then x=ϕ(a)x=\phi(a)x=ϕ(a) for some a∈Ha\in Ha∈H. So x−1=(ϕ(x))−1=ϕ(x−1)∈ϕ(H)x^{-1}=(\phi(x))^{-1}=\phi(x^{-1})\in\phi(H)x−1=(ϕ(x))−1=ϕ(x−1)∈ϕ(H). But x∈Hx\in Hx∈H, so x−1∈Hx^{-1}\in Hx−1∈H, therefore x−1∈ϕ(H)x^{-1}\in\phi(H)x−1∈ϕ(H).
  • If x,y∈ϕ(H)x,y\in \phi(H)x,y∈ϕ(H), then x,y=ϕ(a),ϕ(b)x,y=\phi(a),\phi(b)x,y=ϕ(a),ϕ(b) for some a,b∈Ha,b\in Ha,b∈H. So xy=ϕ(a)ϕ(b)=ϕ(ab)∈ϕ(H)xy=\phi(a)\phi(b)=\phi(ab)\in\phi(H)xy=ϕ(a)ϕ(b)=ϕ(ab)∈ϕ(H) (by homomorphism). Since ab∈Hab\in Hab∈H, xy∈ϕ(H)xy\in\phi(H)xy∈ϕ(H).

(4) If K≤G′K\leq G'K≤G′ then ϕ−1(K)≤G\phi^{-1}(K)\leq Gϕ−1(K)≤G, where ϕ−1(K)={a∈G∣ϕ(a)∈K}\phi^{-1}(K)=\{a\in G|\phi(a)\in K\}ϕ−1(K)={a∈G∣ϕ(a)∈K}.

  • e′∈Ke'\in Ke′∈K implies that e=ϕ−1(e′)∈ϕ−1(K)e=\phi^{-1}(e')\in\phi^{-1}(K)e=ϕ−1(e′)∈ϕ−1(K).
  • If x∈ϕ−1(K)x\in \phi^{-1}(K)x∈ϕ−1(K), then x=ϕ(a)x=\phi(a)x=ϕ(a) for some a∈Ga\in Ga∈G. So x−1=(ϕ(x))−1=ϕ(x−1)∈ϕ−1(K)x^{-1}=(\phi(x))^{-1}=\phi(x^{-1})\in\phi^{-1}(K)x−1=(ϕ(x))−1=ϕ(x−1)∈ϕ−1(K). But x∈Gx\in Gx∈G, so x−1∈Gx^{-1}\in Gx−1∈G, therefore x−1∈ϕ−1(K)x^{-1}\in\phi^{-1}(K)x−1∈ϕ−1(K).
  • If x,y∈ϕ−1(K)x,y\in \phi^{-1}(K)x,y∈ϕ−1(K), then x,y=ϕ(a),ϕ(b)x,y=\phi(a),\phi(b)x,y=ϕ(a),ϕ(b) for some a,b∈Ga,b\in Ga,b∈G. So xy=ϕ(a)ϕ(b)=ϕ(ab)∈ϕ−1(K)xy=\phi(a)\phi(b)=\phi(ab)\in\phi^{-1}(K)xy=ϕ(a)ϕ(b)=ϕ(ab)∈ϕ−1(K) (by homomorphism). Since ab∈Gab\in Gab∈G, xy∈ϕ−1(K)xy\in\phi^{-1}(K)xy∈ϕ−1(K).

Definition for kernel and image of a group homomorphism

Let G,G′G,G'G,G′ be groups and ϕ:G→G′\phi:G\to G'ϕ:G→G′ be a group homomorphism.

ker⁡(ϕ)={a∈G∣ϕ(a)=e′}=ϕ−1({e′})\operatorname{ker}(\phi)=\{a\in G|\phi(a)=e'\}=\phi^{-1}(\{e'\})ker(ϕ)={a∈G∣ϕ(a)=e′}=ϕ−1({e′}) is called the kernel of ϕ\phiϕ.

Facts:

  • ker⁡(ϕ)\operatorname{ker}(\phi)ker(ϕ) is a subgroup of GGG. (proof by previous corollary (4))
  • ϕ\phiϕ is onto if and only if ker⁡(ϕ)={e}\operatorname{ker}(\phi)=\{e\}ker(ϕ)={e} (the trivial subgroup of GGG). (proof forward, by definition of one-to-one; backward, if ϕ(a)=ϕ(b)\phi(a)=\phi(b)ϕ(a)=ϕ(b), then ϕ(a)ϕ(b)−1=e′\phi(a)\phi(b)^{-1}=e'ϕ(a)ϕ(b)−1=e′, so ϕ(a)ϕ(b−1)=e′\phi(a)\phi(b^{-1})=e'ϕ(a)ϕ(b−1)=e′, so ab−1=eab^{-1}=eab−1=e, so a,b=ea,b=ea,b=e, so a=ba=ba=b)
