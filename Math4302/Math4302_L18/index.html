Recall from last lecture, if ϕ:G→G′\phi:G\to G'ϕ:G→G′ is a homomorphism, then G/ker⁡(ϕ)≃ϕ(G)≤G′G/\ker(\phi)\simeq \phi(G)\leq G'G/ker(ϕ)≃ϕ(G)≤G′.

Definition of simple group

GGG is simple if GGG has no proper (H≠G,{e}H\neq G,\{e\}H=G,{e}), normal subgroup.
