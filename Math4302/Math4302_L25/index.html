Rings

Definitions

  • commutative ring: elements a⋅b=b⋅aa\cdot b=b\cdot aa⋅b=b⋅a, ∀a,b∈R\forall a,b\in R∀a,b∈R
  • ring with unity: elements a⋅1=1⋅a=aa\cdot 1=1\cdot a=aa⋅1=1⋅a=a, ∀a∈R\forall a\in R∀a∈R
  • units: elements such that there is a⋅b=1a\cdot b=1a⋅b=1 for some b∈Rb\in Rb∈R.
  • division ring: every element a≠0a\neq 0a=0 has a multiplicative inverse a−1a^{-1}a−1 such that a⋅a−1=1a\cdot a^{-1}=1a⋅a−1=1.
  • field: division ring that is commutative

Integral Domains

Definition of zero divisors

If a,b∈Ra,b\in Ra,b∈R with a,b≠0a,b\neq 0a,b=0 and ab=0ab=0ab=0, then a,ba,ba,b are called zero divisors.

Example of zero divisors

Consider Z6\mathbb{Z}_6Z6​, then 2⋅3=02\cdot 3=02⋅3=0, so 222 and 333 are zero divisors.

And 4⋅3=04\cdot 3=04⋅3=0, so 444 and 333 are zero divisors.

If aaa is a unit, then aaa is not a zero divisor.

ab=0  ⟹  a−1ab=0  ⟹  1b=0  ⟹  b=0ab=0\implies a^{-1}ab=0\implies 1b=0\implies b=0ab=0⟹a−1ab=0⟹1b=0⟹b=0.

Note

If an element is not unit, it may not be a zero divisor.

Consider R=ZR=\mathbb{Z}R=Z and 222 is not a unit, but 222 is not a zero divisor.

Definition of integral domain

A commutative ring with unity is called a integral domain (or just a domain) if it has no zero divisors.

Example of integral domain

Z\mathbb{Z}Z is a integral domain.

Any field is a integral domain.

Corollaries of integral domain

If RRR is a integral domain, then we have cancellation property ab=ac,a≠0  ⟹  b=cab=ac,a\neq 0\implies b=cab=ac,a=0⟹b=c.

Units with multiplication forms a group

If RRR is a ring with unity, then the units in RRR forms a group under multiplication.

Proof

if a,ba,ba,b are units, then ababab is a unit (ab)−1=b−1a−1(ab)^{-1}=b^{-1}a^{-1}(ab)−1=b−1a−1.

In particular, non-zero elements of any field form an abelian group under multiplication.

Example

Consider Zp\mathbb{Z}_pZp​ field, then ({1,2,⋯ ,p−1},⋅)(\{1,2,\cdots,p-1\},\cdot)({1,2,⋯,p−1},⋅) forms an abelian group of size p−1p-1p−1.

Consider Z5\mathbb{Z}_5Z5​, then we have a group of size 444 under multiplication.

  • 111 has order 1
  • 222 has order 4 2,4,3,12,4,3,12,4,3,1.
  • 333 has order 4 3,4,2,13,4,2,13,4,2,1.
  • 444 has order 2 4,14,14,1.

Therefore Z5≃Z4\mathbb{Z}_5\simeq \mathbb{Z}_4Z5​≃Z4​.
