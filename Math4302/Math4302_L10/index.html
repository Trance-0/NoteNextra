Group homomorphism

Recall the kernel of a group homomorphism is the set

ker⁡(ϕ)={a∈G∣ϕ(a)=e′}\operatorname{ker}(\phi)=\{a\in G|\phi(a)=e'\}ker(ϕ)={a∈G∣ϕ(a)=e′}

Example

Let ϕ:(Z,+)→(Zn,+)\phi:(\mathbb{Z},+)\to (\mathbb{Z}_n,+)ϕ:(Z,+)→(Zn​,+) where ϕ(k)=kmod  n\phi(k)=k\mod nϕ(k)=kmodn.

The kernel of ϕ\phiϕ is the set of all multiples of nnn.

Theorem for one-to-one group homomorphism

ϕ:G→G′\phi:G\to G'ϕ:G→G′ is one-to-one if and only if ker⁡(ϕ)={e}\operatorname{ker}(\phi)=\{e\}ker(ϕ)={e}

If ϕ\phiϕ is one-to-one, then ϕ(G)≤G′\phi(G)\leq G'ϕ(G)≤G′, GGG is isomorphic ot ϕ(G)\phi(G)ϕ(G) (onto automatically).

If AAA is a set, then a permutation of AAA is a bijection f:A→Af:A\to Af:A→A.

Odd and even permutations

Definition of transposition

A σ∈Sn\sigma\in S_nσ∈Sn​ is a transposition is a two cycle σ=(ij)\sigma=(i j)σ=(ij)

Fact: Every permutation in SnS_nSn​ can be written as a product of transpositions. (may not be disjoint transpositions)

Example of a product of transpositions

Consider (1234)=(14)(13)(12)(1234)=(14)(13)(12)(1234)=(14)(13)(12).

In general, (i1,i2,⋯ ,im)=(i1im)(i2im−1)(i3im−2)⋯(i1i2)(i_1,i_2,\cdots,i_m)=(i_1i_m)(i_2i_{m-1})(i_3i_{m-2})\cdots(i_1i_2)(i1​,i2​,⋯,im​)=(i1​im​)(i2​im−1​)(i3​im−2​)⋯(i1​i2​)

This is not the unique way.

(12)(34)=(42)(34)(23)(12)(12)(34)=(42)(34)(23)(12)(12)(34)=(42)(34)(23)(12)

But the parity of the number of transpositions is unique.

Theorem for parity of transpositions

If σ∈Sn\sigma\in S_nσ∈Sn​ is written as a product of transposition, then the number of transpositions is either always odd or even.

Definition of odd and even permutations

σ\sigmaσ is an even permutation if the number of transpositions is even.

σ\sigmaσ is an odd permutation if the number of transpositions is odd.
