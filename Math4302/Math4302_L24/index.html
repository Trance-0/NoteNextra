Rings

Definition of ring

A ring is a set RRR with binary operation +++ and ⋅\cdot⋅ such that:

  • (R,+)(R,+)(R,+) is an abelian group.
  • Multiplication is associative: (a⋅b)⋅c=a⋅(b⋅c)(a\cdot b)\cdot c=a\cdot (b\cdot c)(a⋅b)⋅c=a⋅(b⋅c).
  • Distribution property: a⋅(b+c)=a⋅b+a⋅ca\cdot (b+c)=a\cdot b+a\cdot ca⋅(b+c)=a⋅b+a⋅c, (b+c)⋅a=b⋅a+c⋅a(b+c)\cdot a=b\cdot a+c\cdot a(b+c)⋅a=b⋅a+c⋅a. (Note that ⋅\cdot⋅ may not be abelian, may not even be a group, therefore we need to distribute on both sides.)
Note

a⋅b=aba\cdot b=aba⋅b=ab will be used for the rest of the sections.

Examples of rings

(Z,+,∗)(\mathbb{Z},+,*)(Z,+,∗), (R,+,∗)(\mathbb{R},+,*)(R,+,∗) are rings.

(2Z,+,⋅)(2\mathbb{Z},+,\cdot)(2Z,+,⋅) is a ring.

(Mn(R),+,⋅)(M_n(\mathbb{R}),+,\cdot)(Mn​(R),+,⋅) is a ring.

(Zn,+,⋅)(\mathbb{Z}_n,+,\cdot)(Zn​,+,⋅) is a ring, where a⋅b=a∗bmod  na\cdot b=a*b\mod na⋅b=a∗bmodn.

e.g. in Z12,4⋅8=8\mathbb{Z}_{12}, 4\cdot 8=8Z12​,4⋅8=8.

Tip

If (R+,⋅)(R+,\cdot)(R+,⋅) is a ring, then (R,⋅)(R,\cdot)(R,⋅) may not be necessarily a group.

Definition of commutative ring

A ring (R,+,⋅)(R,+,\cdot)(R,+,⋅) is commutative if a⋅b=b⋅aa\cdot b=b\cdot aa⋅b=b⋅a, ∀a,b∈R\forall a,b\in R∀a,b∈R.

Example of non commutative ring

(Mn(R),+,⋅)(M_n(\mathbb{R}),+,\cdot)(Mn​(R),+,⋅) is not commutative.

A ring RRR has unity element if there is an element 1∈R1\in R1∈R such that a⋅1=1⋅a=aa\cdot 1=1\cdot a=aa⋅1=1⋅a=a, ∀a∈R\forall a\in R∀a∈R.

Definition of division ring

If every a≠0a\neq 0a=0 in RRR has a multiplicative inverse (is a unit), then RRR is called a division ring.

Definition of field

A commutative division ring is called a field.

Example of field
